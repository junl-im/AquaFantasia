# AquaFantasia v2.1.51

## v2.1.51 변경사항

- 우측 상단 메뉴는 뒤쪽 큰 프레임을 제거하고, 아이콘 크기에 맞춘 반투명 1중 셀만 유지합니다.
- 하단 홈/가방/퀘스트/지도 아이콘은 버튼 내부에서만 보이도록 클리핑과 contain을 다시 고정했습니다.
- 가방/퀘스트/지도/상점 페이지의 중복 제목 잔상, 카드 겹침, 상점 글/가격 버튼 가림을 한 번 더 정리했습니다.
- 개척 항로 조사/후보지/출항 후보지 영역은 공통 아쿠아 카드 스킨으로 묶어 덩그러니 보이는 텍스트를 줄였습니다.
- 낚시 화면은 준비 단계와 릴링 단계를 분리하고, 입질 후 본 게임 콕핏이 좌우 화면 밖으로 밀리지 않도록 inset 기반 중앙 레이아웃을 보강했습니다.
- 낚시의 바다물길 표기는 하나만 유지하고, 구형 프레임/소품/게이지 이미지 조각은 계속 차단합니다.
- 타일 픽셀 크기는 유지합니다. v2.1.51는 다이아몬드 터치 점수만 `0.940`으로 소폭 조정하며, 타일 픽셀 축소는 세이브 좌표, 건물 footprint, NPC 이동, 충돌 판정, 카메라 경계 마이그레이션 전까지 보류합니다.
- `node_modules`는 CI 작업환경에서는 허용하지만, Git 추적 및 full/patch ZIP 포함은 계속 차단합니다.

## 패키징 규칙

- 루트 Markdown은 `README.md` 하나만 유지합니다.
- 루트 `APP_VERSION` 파일은 만들지 않습니다.
- `node_modules`, `dist`, `reports`, `backup`, `logs`, `_NOTES`, `APP_VERSION`, `*.log`는 배포 ZIP에 넣지 않습니다.
