# AquaFantasia Patch 21 - v3.2 Visual Atlas Mega Update

## 목표
자잘한 기능 추가보다 대형 점검과 디자인 정리를 우선했습니다. v3.1을 기준으로 UI 시선 흐름, 아트 에셋 부족, 모바일 플레이 집중도, 저장/검증 기준을 다시 점검했습니다.

## 주요 변경
- Visual Atlas 3.2 커맨드 센터 추가
- v3.2 SVG 아트팩 14종 추가
- 신규 어종 15종 추가, 총 70종
- 추천 행동/지역/도감/성능을 한 카드에 압축
- 카카오 무팝업 모바일 게임 셸 유지
- 전체화면 첫 입력 자동 시도, 뒤로가기 종료 확인 흐름 유지
- Service Worker, manifest, package, validator v3.2 갱신

## 해결한 문제
- v3.1 화면의 시선 흐름이 Director Cut, Ocean Director, Focus UI로 분산되던 문제를 Visual Atlas가 최상위 요약으로 정리
- 지역별 아트가 아직 카드 수준에 머물던 문제를 panorama 에셋으로 보강
- fish DB 확장 후 검증 기준을 70종 이상으로 강화
