# AquaFantasia v2.1.21

## v2.1.21 변경사항

- 버전을 `2.1.21`로 동기화했습니다.
- `v2121-ui-continuity-polish-root`를 추가해 v2.1.20의 오프닝 영상, 로그인 유지 토글, 플레이어 방향 매핑 고정값은 유지하고 화면 전환/메뉴/모달/낚시 UI 연속성을 다듬었습니다.
- 마을 오프닝 영상에 안전한 `바로 마을로` 스킵 버튼을 추가했습니다. 영상이 막히거나 오류가 나도 포스터/아쿠아 로딩 연출로 넘어가도록 `v2121-opening-video-ready` / `v2121-opening-video-fallback` 상태를 추가했습니다.
- 마을 로딩 중에는 HUD, 개척 바, 우측 상단 조작바, 조이스틱, 우측 하단 메뉴바가 먼저 튀어나오지 않도록 v2.1.20 로딩 차단 규칙을 유지했습니다.
- 왼쪽 위 HUD와 HUD 아래 개척 바 간격을 다시 미세 조정하고, 우측 상단 `+ / -`, `건설 / 원점`, `상점 / 출항` 조작바의 아이콘 잘림을 줄이도록 크기를 아주 조금 보정했습니다.
- 우측 하단 `홈 / 가방 / 퀘스트 / 지도` 메뉴바는 눌렀을 때 버튼이 밀리지 않도록 폭/높이/아이콘/라벨 크기를 고정했습니다.
- 가방, 퀘스트, 지도, 상점 같은 메뉴 페이지에 공통 페이지 헤더를 부여했습니다. 메뉴 HUD는 따라오지 않고, 중앙 아쿠아 카드 안에서 제목/설명/본문이 한 흐름으로 보이도록 정리했습니다.
- 캐릭터 정보창, 건물 정보창, 건설창, 개척창, 종료 팝업의 공통 아쿠아 카드 톤, 딤/블러, 우측 상단 X 닫기, 글씨 번짐 방지 규칙을 다시 묶었습니다.
- 낚시 화면에서 릴 패널, 감기/풀기 콘솔, 캐스팅 버튼, 터치존이 좌우로 밀리지 않도록 중앙 고정 규칙을 강화했습니다. 낚시 소품은 화면을 덮지 않는 약한 장식 수준으로 제한했습니다.
- 조이스틱은 기본 위치만 중앙이며, 조작 중에는 `--v2121-joystick-transform` 값으로 정상 이동하도록 v2.1.20 동작을 이어받았습니다.
- 현재 승인된 `이 기기에서 로그인 유지` 토글 스타일은 고정했습니다. PNG/이미지 배경을 다시 입히지 않고, 작고 얇은 CSS 아쿠아 토글과 `✓` 체크만 유지합니다.
- 플레이어 8방향 32프레임과 v2.1.20 방향 매핑 보정은 건드리지 않고 검증 고정만 유지했습니다.

## 검증 기준

- v2.1.21 작업본에서 `npm run validate`가 통과해야 합니다.
- v2.1.21 통파일 ZIP을 새로 풀어 `npm run validate`가 통과해야 합니다.
- v2.1.20 통파일에 v2.1.21 패치 ZIP을 덮어쓴 뒤 `npm run validate`가 통과해야 합니다.
- `package.json`, `package-lock.json`, `APP_VERSION`, service worker cache, offline badge/version이 `2.1.21`로 동기화되어야 합니다.
- 루트 Markdown 파일은 이 `README.md` 하나만 유지합니다.
- ZIP에는 `node_modules`, `dist`, `reports`, `AquaFantasia_backup_v1`, `.log`, `*_NOTES.md`를 넣지 않습니다.
- 금지 registry/internal 문자열은 포함하지 않습니다.
- 캐릭터 방향 고정 토큰과 v2.1.20 플레이어 방향 보정 매핑은 유지합니다.
- 현재 승인된 로그인 유지 토글 스타일은 이미지 배경 없이 유지합니다.
- v2.1.10 신규 에셋 278개와 v2.1.18~v2.1.20 캐릭터/타일/오프닝 에셋은 유지합니다.
