# AquaFantasia Asset Pack

## Included assets

This ZIP contains the generated assets currently available in this conversation, organized for game production:

- `01_environment_background/`
  - First village town square background, mobile portrait 9:16.
- `02_buildings_png_transparent/`
  - Fisherman House
  - Fish Market
  - Village Inn
  - Harbor Office
  - Fishing Guild
  - Aquarium
  - Blacksmith
  - Storage Warehouse
- `03_path_tiles_png_transparent/`
  - Modular stone path tileset.
- `98_original_generated_png_backup/`
  - Unmodified generated PNG backups.
- `99_docs/`
  - Art bible and manifest.

## Art Bible

Genre: Fantasy Ocean RPG  
Style: Stylized SD  
Camera: Quarter View  
Mood: Warm, Bright, Adventure  
Rendering: 2.5D Hand Painted  
Target: Mobile Portrait / HD

## Visual rules

Recommended materials:
- Blue crystal
- Coral
- Seashell
- Wood
- Bright beige stone
- Warm lantern light

Avoid:
- Heavy gold
- Heavy black
- Lineage-like style
- Generic mass-market MMORPG look

## Production note

The isolated building/path PNGs were post-processed into RGBA transparency from the generated checker/white background. The `98_original_generated_png_backup` folder keeps untouched originals in case you want to do manual cutout or re-export.
