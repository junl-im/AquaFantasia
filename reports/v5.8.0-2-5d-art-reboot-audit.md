# v5.8.0 2.5D Art Reboot Audit

## 문제 진단

사용자 피드백 기준으로 v5.7.0 이후 배경은 개선됐지만, UI는 여전히 누적 패치 흔적이 많이 남아 있었습니다.

- 배경은 원화풍인데 카드/버튼은 플랫하거나 반투명 유리 느낌에 머무름
- 첫 실행 가이드가 너무 큰 직사각형 패널처럼 보여 모바일 게임 감성이 부족함
- 하단 네비게이션이 기능성 바처럼 보이고 게임용 독 느낌이 약함
- 개발용 텍스트가 일부 환경에서 다시 나타날 수 있음
- 여러 패치의 CSS가 섞여 같은 화면 안에서 스타일 위계가 흔들림

## 해결 방향

v5.8.0은 전체 이미지를 한 번에 DCC 원화 에셋으로 교체하는 대신, 실제 런타임에서 바로 보이는 UI 표면을 2.5D 규칙으로 통일했습니다.

- 상단 하이라이트 + 하단 음영 + 내부 엣지 + 두꺼운 라운드
- primary 버튼은 노랑/민트/하늘/블루 그라데이션으로 통일
- 패널은 깊은 해양색 기반에 유리 하이라이트만 얹는 방식
- 하단 네비는 검은 바가 아니라 떠 있는 모바일 게임 독으로 변경
- 동적 DOM에도 `MutationObserver`로 재장식 적용

## 안정성

기능 함수 오버라이드는 하지 않고 CSS/DOM 보강 위주로 작성했습니다. 기존 v5.5.4 Stack Guard, v5.5.5 Auto Cache Sweep, v5.6.1 UI Cleanup, v5.7 Water Art와 함께 순차 적용됩니다.
