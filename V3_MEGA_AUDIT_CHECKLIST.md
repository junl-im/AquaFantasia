# v3.0 Mega Audit Checklist

## 모바일 첫 화면
- [ ] v3.0 배지가 표시된다
- [ ] 오션 디렉터 패널이 보인다
- [ ] 추천 행동 버튼이 출석/가방/장비/추천 지역 상황에 맞게 바뀐다
- [ ] 스마트 독으로 낚시/가방/장비/지도 이동이 된다

## 카카오/모바일
- [ ] 카카오에서 외부 브라우저 권장 팝업이 뜨지 않는다
- [ ] 카카오 안에서 모바일 게임 셸이 바로 적용된다
- [ ] 첫 터치 후 가능한 브라우저에서 전체화면 시도가 된다
- [ ] 뒤로가기 버튼을 누르면 종료 확인창이 뜬다

## 디자인/에셋
- [ ] 새 배경/카드/낚시 스테이지 SVG가 로드된다
- [ ] 저사양 모드에서 과도한 애니메이션이 줄어든다
- [ ] 지역 카드가 추천/잠금 상태를 시각적으로 구분한다

## 게임 루프
- [ ] 새 포획분은 판매 전 골드가 바로 증가하지 않는다
- [ ] 판매 추천/보관 추천이 작동한다
- [ ] 월드맵 잠금 안내가 현재 점수/필요 점수를 제대로 표시한다
- [ ] 신규 어종이 도감/포획 흐름에 섞인다

## 배포
- [ ] npm run validate 통과
- [ ] GitHub Actions Pages 배포 성공
- [ ] 모바일 PWA 캐시가 v3.0으로 갱신된다
