# Patch 04 - v1.5 Equipment UX

AquaFantasia v1.5는 그래픽 보정 2차 패치와 장비 성장 시스템을 함께 적용한 패치입니다.

## 핵심 추가

- 신규 화면: `장비 공방`
- 낚싯대 4종 추가
  - 기본 낚싯대
  - 카본 낚싯대
  - 수정 낚싯대
  - 용린 낚싯대
- 미끼 4종 추가
  - 기본 미끼: 무제한
  - 반짝 미끼
  - 진주 미끼
  - 용궁 미끼
- 장비 효과가 실제 낚시에 반영됩니다.
  - 릴 파워
  - 장력 제어
  - 희귀 어종 가중치
  - 입질 대기 시간
  - 판매가 보너스
  - 경험치 보너스

## UI / 그래픽 보정

- 마을 화면에 현재 장착 장비 요약 표시
- 하단 메뉴에 장비 탭 추가
- 장비 카드 홀로그램 쉬머 효과 추가
- 낚시 화면에 심도 미터와 미끼 투척 광원 효과 추가
- 화면이 백그라운드로 전환되면 애니메이션 일시 정지
- 저사양 기기에서는 기본 그래픽 모드를 자동 절전으로 선택

## 저장 호환성

- 새 저장 키: `aqua_v1.5`
- 이전 저장 키 호환 유지: `aqua_v1.1` ~ `aqua_v1.4`
- 기존 유저는 기본 낚싯대와 기본 미끼를 자동 지급받습니다.
