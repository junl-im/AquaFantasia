# AquaFantasia v8.4.0 Fullscreen Fishing UI Polish

## 목표
v8.3.0에서 전체 UI가 작아지고 낚시터가 메뉴 구성도 안에 갇혀 보이던 문제를 우선 수정한다. 낚시터는 전체 화면 게임 배경으로 복구하고, 하단 네비/랭킹/카카오 풀스크린 재진입/캐릭터 선명도 문제를 함께 정리한다.

## 주요 수정
- 낚시 화면을 v13 전체 구성도 이미지에서 분리하고 `v840-fishing-screen` 실시간 PixiJS 풀스크린 스테이지로 재구성.
- 낚싯대 / 미끼 / 오늘 목표 카드와 관련 핫스팟 제거.
- `낚시 시작` CTA는 유지하고 하단 최근 포획 패널 위에 크게 배치.
- 최근 포획 패널을 하단 메뉴 바로 위로 이동.
- 4x 업스케일 배경 에셋을 `public/assets/v84/bg`에 추가하고 낚시 배경으로 연결.
- 캐릭터 스프라이트를 4x 보정/선명화해서 `public/assets/v84/characters/chibi_fisher_01_hd.png`로 연결.
- 하단 탭 네비를 투명 hotspot 방식에서 실제 아이콘 버튼형 네비로 복구.
- v13 메뉴 화면은 106.5%로 소폭 확대해서 UI가 작아 보이는 문제 완화.
- 메뉴 우측 상단의 불필요해 보이는 2개 버튼 영역을 `v13-top-action-cleaner`로 시각 제거.
- 랭킹은 가짜 유저 데이터를 제거하고 현재 저장 기록 기준의 내 계정만 표시.
- 카카오/인앱 브라우저에서 메뉴 클릭/액션/재진입 시 몰입형 세로 화면을 재요청.
- 세로 고정 정책은 유지. 가로 전환 레이아웃은 추가하지 않음.

## 검증
- `npm run typecheck`
- `npm run validate`
- `npm run build`

모두 통과.

## 남은 과제
- 장비/상점/미션/가방/도감은 아직 v13 구성도 이미지를 바탕으로 한 안정화 단계다. 다음 단계에서 9-slice 파츠/실제 DOM 데이터 바인딩으로 완전히 분해하는 것이 맞다.
- Firebase 서버 랭킹은 현재 인증/저장 규칙 연결 전이므로 로컬 저장 기록만 표시한다. Firestore 컬렉션 설계와 보안 규칙이 붙으면 실제 전체 유저 랭킹으로 확장 가능하다.
