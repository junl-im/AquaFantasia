# AquaFantasia Patch Notes v2.6 - Balance Core

## 목표
UI 정리 이후 실제 플레이에 도움이 되는 성장 밸런스와 지역 추천 시스템을 추가했습니다.

## 변경 사항
- Balance Coach 카드 추가: 추천 지역, 전투력, 다음 행동을 한눈에 표시
- 지역별 권장 전투력/레벨/위험도/보상 배율 추가
- 지역 카드에 준비도 미터 표시
- 초반 10마리 구간 포획 보정으로 첫 플레이 이탈 감소
- 높은 지역은 보상이 커지지만 장력 상승도 커지는 위험/보상 구조 적용
- 낚싯대 구매 비용과 강화 비용 곡선 완화
- 미끼 가격/묶음 수량 조정으로 소모 부담 감소
- 장비 화면에 추천 지역 정보 추가
- 저장 키 aqua_v2.6 추가, 기존 v2.5 이하 저장 데이터 호환
- PWA 캐시 및 GitHub Actions 검증 기준 v2.6 갱신

## 적용 이유
이전 버전은 기능과 화면이 많아졌지만, 유저가 어디서 낚시하고 무엇을 강화해야 하는지 판단하기 어려웠습니다. v2.6은 실제 플레이 루프를 포획 → 판매 → 강화 → 지역 확장으로 명확히 만드는 데 집중했습니다.
