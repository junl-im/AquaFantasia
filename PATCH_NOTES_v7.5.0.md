# Aqua Fantasia v7.5.0 UI Rescue + Mission Expansion

## 핵심 수정
- 시작 화면의 `이 기기에서 로그인 유지` 영역을 원화에 박힌 글자/체크와 분리하고, 새 2.5D 토글 버튼으로 재배치했습니다.
- 마을/장비/도감/상점/미션 화면은 세로 스크롤이 가능하도록 수정했습니다.
- 낚시터 화면은 기존처럼 고정형 액션 화면으로 유지합니다.
- 상점/미션 텍스트 가독성을 강화했습니다.
- 미션을 출석/오늘/성장/도감/보스/콤보/장비/상점/수역/숙련/깜짝 이벤트 계열로 대폭 확장했습니다.
- 사용자가 제공한 에셋풍을 유지하되, 버튼/프레임은 줄무늬 없는 깨끗한 2.5D UI로 재가공했습니다.
