# AquaFantasia v2.1.28

## v2.1.28 변경사항

- 플레이어 3시 방향이 9시처럼 보이던 문제를 실제 에셋 단계에서 수정했습니다.
  - `public/assets/v2128/characters/player`에 보정된 8방향 32프레임 세트를 추가했습니다.
  - 3시 방향은 v2.1.18 서쪽 프레임을 좌우 반전한 실제 오른쪽 보기 프레임을 사용합니다.
  - 방향 계산 함수는 건드리지 않고 플레이어 전용 시각 프레임만 고정했습니다.
- 걷기 모션이 떨림처럼 보이지 않도록 프레임 전환 속도와 좌우 흔들림을 더 낮췄습니다.
- 마을 우측 하단 메뉴바가 마을 화면에서만 이상하게 보이는 문제를 줄이기 위해 위치/크기/아이콘/눌림 상태를 `v2128-bottom-nav`로 재고정했습니다.
- 우측 상단 메뉴바의 외곽 테두리를 투명화하고, 아이콘 간격을 조금 더 넓혔습니다.
- 가방/퀘스트/지도/상점/건설/건물상세/캐릭터 정보/개척/종료 팝업의 아쿠아 카드 스타일을 다시 정리했습니다.
- 낚시 화면은 릴 패널, 캐스팅 버튼, 최근 포획, 장비바, HUD 위치를 `v2128-fishing-upgrade-screen`에서 다시 안정화했습니다.
- 오프닝 영상의 구형 테두리/프레임 차단 규칙은 유지했습니다.
- `이 기기에서 로그인 유지` 토글 스타일은 기존 고정 상태를 유지했습니다.

## 검증

- `npm run validate`는 `tools/check-v2128-direction-ui-fishing-audit.mjs`를 실행합니다.
- 검증 항목:
  - v2.1.28 버전 동기화
  - CSS 괄호/중괄호/대괄호 균형
  - v2128 플레이어 8방향 32프레임 존재
  - v2128 동쪽 프레임이 기존 v2118 동쪽 프레임과 같지 않은지 확인
  - 동/서 방향 매핑 고정
  - 조이스틱 v2128 transform 변수
  - 우측 상단/하단 메뉴 클래스
  - 낚시 v2128 안정화 클래스
