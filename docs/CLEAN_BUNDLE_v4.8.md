# AquaFantasia v4.8 Clean Bundle Guide

- index.html, sw.js, manifest, offline.html
- Firebase rules/config
- data/fish.json
- 현재 런타임에서 참조되는 assets
- src, tools, package/vite/tsconfig
- 최신 v4.8 문서

- 오래된 PATCH_NOTES
