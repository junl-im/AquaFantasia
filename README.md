# AquaFantasia v1.0.1

모바일 웹 낚시 게임 `AquaFantasia`의 새 버전 네이밍 시작 패치입니다.

이제 버전은 `1.0.1 → 1.0.2 → ... → 1.1.0` 방식으로 진행합니다.

## 이번 패치 목표

- 투명 프레임 뒤가 뻥 뚫려 보이는 문제 수정
- 테두리 크기와 내부 배경 크기를 맞춰 UI가 작아 보이는 느낌 완화
- 마을 상단 로고/HUD 프레임을 더 예쁘게 정리
- 하단 메뉴바 위아래 여백 축소
- 하단 검은 띠 제거
- 콤보 배지와 낚시 시작 버튼 겹침 방지
- 버튼형 UI는 아담하게 줄이되 렌더드 버튼 느낌 유지
- 첨부한 수중 셰이더 이미지팩을 WebGL 배경 텍스처로 실제 사용
- `README.md` 단일 문서 정책 유지

## 주요 변경

### UI 프레임

프레임 PNG가 투명 중심부를 가진 경우, 뒤 배경이 그대로 뚫려 보이지 않도록 같은 크기 안쪽에 그라디언트 fill 레이어를 깔았습니다.

- 큰 패널: 프레임 + 내부 수중 패널 fill
- 카드/상점/장비/미션/도감/랭킹: 카드 프레임 + 내부 fill
- 버튼: 기존 2.5D 렌더드 버튼 질감 유지, 높이와 패딩 축소

### 하단 메뉴바

- 하단 전체 프레임 방향 유지
- 위아래 여백 축소
- 하단 검은 띠가 보이지 않도록 bottom 0 기준으로 재배치
- 현재 메뉴 표시는 과한 테두리가 아니라 작은 빛/진주 표시만 유지

### 낚시 화면

- 콤보 배지를 상단 쪽으로 옮겨 낚시 시작 버튼과 겹치지 않게 수정
- 최근 포획, 시작 버튼, 릴 게이지, 하단 메뉴 간격 재조정
- 릴 게이지와 최근 포획 패널에도 내부 fill을 추가해 투명 프레임 느낌 제거

### WebGL 수중 레이어

새로 첨부한 `underwater_webgl_shader_images_vertical` 에셋을 `public/assets/v101/water`로 변환했습니다.

- WebGL 셰이더가 배경 이미지를 sampler texture로 읽어 수중 왜곡/카스틱/빛줄기와 섞습니다.
- WebGL 미지원 기기에서는 같은 이미지를 CSS 배경으로 사용합니다.
- 메뉴별/수역별로 다른 수중 이미지를 사용합니다.
