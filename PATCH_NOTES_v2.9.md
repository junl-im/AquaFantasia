# Aqua Fantasia Patch 18 - v2.9 Art UI Mega Refresh

## 핵심
- 원화급 느낌을 목표로 한 SVG 기반 고급 배경/프레임/낚시 스테이지 추가
- 모바일 화면 높이 자동 보정, 카카오 인앱브라우저 자동 최적화 모드
- 카카오 외부 브라우저 권장 팝업 제거
- 첫 터치 시 전체화면/화면 켜짐 유지 자동 시도
- 모바일 뒤로가기 버튼 입력 시 바로 나가지 않고 종료 확인 모달 표시
- 기존 저장 데이터 v2.8 이하 호환 유지, 새 저장 키 `aqua_v2.9` 추가
- 인라인 onclick 핸들러를 window에 명시적으로 노출해 Firebase module 스크립트 환경 안정화
- PWA/Service Worker/Actions 검증 기준 v2.9 갱신

## 참고
브라우저 보안 정책상 일반 웹페이지가 사용자 동작 없이 외부 브라우저를 강제 실행하거나 전체화면으로 진입하는 것은 제한됩니다. v2.9는 팝업을 없애고 카카오 내부에서 바로 플레이 가능한 몰입형 셸을 자동 적용하며, 첫 터치 사용자 활성화 시점에 전체화면을 자동 시도합니다.
