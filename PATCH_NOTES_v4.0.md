# Aqua Fantasia v4.0 Boss Action Fishing

대형 낚시 전투 업데이트입니다. v3.9의 터치 링/릴 타이밍 위에 보스 패턴, 줄 풀기, 차지 릴, 돌진 회피, 물고기 궤적 추적, 낚시 세부 설정을 추가했습니다.

## 핵심
- 보스 텔레그래프: 붉은 돌진, 장력 폭주, 카운터 창
- 신규 액션: 줄 풀기 / 차지 릴 / 돌진 회피
- 물고기 궤적: 중앙 타깃 구간 근처에서 릴 감기 효율 증가
- 성능 옵션: 디테일, 조작 안내, 낚시 성능 전환
- v4.0 SVG 전투 에셋 추가
