# Aqua Fantasia Patch 07 - v1.8 Forge Visual Max

## 핵심 업데이트

- 장비 공방에 포지 강화 패널 추가
- 낚싯대 강화 레벨 +0~+10 시스템 추가
- 강화 성공률, 비용, 실패 보정(pity) 추가
- 강화 효과가 릴 파워, 장력 제어, 희귀도, 판매가, 경험치에 실제 반영
- 장비 점수와 성장 투자율 UI 추가
- 강화 성공/실패 Web Animations API 기반 파티클 연출 추가
- Scheduler API 지원 브라우저에서 무거운 화면 렌더링을 우선순위 작업으로 분리
- CSS @property 기반 포지 오라 애니메이션 추가
- v1.1~v1.7 저장 데이터 호환 유지, 새 저장 키 aqua_v1.8 추가
- Service Worker 캐시 버전 v1.8.0으로 갱신

## 밸런스 메모

- 강화는 실패해도 장비가 파괴되지 않습니다.
- 실패 시 다음 강화 성공률이 소폭 보정됩니다.
- 강화 비용은 레벨이 오를수록 증가합니다.
- 초반에는 기본 낚싯대 강화보다 카본 낚싯대 구매 후 강화가 효율적입니다.

## 테스트 체크

1. 기존 저장 데이터가 유지되는지 확인
2. 장비 공방에서 강화 패널 표시 확인
3. 골드 부족 시 성장 추천 토스트 확인
4. 강화 성공 시 +레벨과 스탯 증가 확인
5. 실패 시 장비 파괴 없이 보정 메시지 확인
6. 새로고침 후 강화 레벨 유지 확인
