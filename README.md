# AquaFantasia v1.1.0 Layout + Color UI Polish

이번 패치는 v1.0.10 이후 화면이 옆으로 밀리거나 UI가 화면 밖으로 나가 보이는 문제를 우선 해결한 안정화 패치입니다.

## 핵심 변경
- 스와이프 메뉴 이동에서 pointer/touch 중복 처리 제거
- 낚시 화면은 계속 스와이프 메뉴 이동 제외
- 하단 메뉴바를 세로 게임 셸 중앙에 고정하고, 100vw 기준 밀림 방지
- 하단 메뉴 아이콘은 크게 유지하고 프레임만 더 슬림하게 조정
- 메뉴/카드/버튼/텍스트는 100vw가 아니라 게임 화면 폭 기준으로 containment 처리
- 상점, 미션, 랭킹, 장비, 가방, 도감 카드의 글씨/버튼 넘침 추가 방지
- 미션 진행 게이지바 색감/가독성 정리
- 랭킹 글씨 대비와 한 줄 기록 가독성 보강
- 낚시 화면의 낚시 시작, 최근 포획, 릴 패널, 콤보 위치 분리
- 수중 WebGL 효과는 유지하되 UI를 묻지 않도록 알파/대비 조정
- 문서는 README.md 하나만 유지

## 검증
```bash
npm run validate
npm run typecheck
npm run build
npm run audit
```
