# Aqua Fantasia - 고퀄리티 낚시 게임 UI & 스프라이트 키트

**테마**: 판타지 낚시 (심해 블루 + 틸 + 골드 마법 느낌)
**스타일**: 다크 판타지 + 수중 판타지 (이전 UI 키트와 일관된 고급스러운 느낌)

## 포함 내용

### 1. Icons (약 20개+)
**낚시 핵심**
- `fishing_rod.png` (낚시대)
- `bobber.png` (찌)
- `bait_worm.png` (미끼)
- `water_ripple.png` (물결)

**물고기 6종** (희귀도별 컬러)
- fish_1 ~ fish_6 (일반 → 신화급)

**지역 아이콘**
- region_lake, region_river, region_deep_sea, region_dragon_palace

**미니게임**
- tension_gauge (장력 게이지)
- reel_progress (릴 진행도 바)

**기타**
- rod_upgrade, inventory_fish, sell, forge

### 2. Fishing Minigame 전용 요소
- `tension_meter_80px.png`, `tension_meter_120px.png` (수직 장력 게이지)
- `reel_bar_160px.png`, `reel_bar_220px.png` (릴 진행 바)
- `bobber_large.png` (64x64 큰 찌 + 물결 효과)

### 3. Panels (8개)
- 낚시 인벤토리
- 장비 강화
- 지역 선택
- 미션
- 상점
- 보스 낚시
- 랭킹
- 설정

모든 패널은 타이틀바 + 골드 베벨 스타일 적용

## 사용 추천

**낚시 미니게임**
- `reel_bar_*.png` + `tension_meter_*.png` 조합으로 긴장감 있는 리링 미니게임 제작 가능
- `bobber_large.png`로 캐스팅/대기 연출

**인벤토리 / 상점**
- fish 아이콘들로 잡은 물고기 표시
- `sell`, `forge` 아이콘으로 상호작용

**지역 선택**
- region_* 아이콘으로 월드맵 또는 지역 선택 UI 구성
