# AquaFantasia v8.1.0 Asset Fit + Start Keep Button Audit

## 목적

v8.0.0에서 사용자가 지적한 문제를 기준으로, 새로 첨부된 `AquaFantasia_asset_pack_split_plus.zip`와 `AquaFantasia_v12_exact_reference_fishing_style_assets.zip`의 에셋을 실제 런타임 UI에 맞게 재분류하고 반영했다.

핵심 기준은 다음과 같다.

- SVG/벡터 느낌 제거
- 2.5D/3D 렌더드 PNG/WebP 우선
- 글씨가 구워진 이미지 위에 DOM 글씨가 겹치지 않도록 빈 버튼/빈 패널 에셋 사용
- 첫 시작 화면의 `이 기기에서 로그인 유지`를 삭제하지 않고 실제 버튼형 UI로 복구
- 장비 / 상점 / 미션 화면에서 카드와 텍스트가 잘 보이도록 모바일 세로 레이아웃 보강

## 반영 에셋 폴더

새 런타임 에셋은 다음 위치에 정리했다.

```text
public/assets/v12/
├─ bg/
├─ buttons/
├─ characters/
├─ fish/
├─ fx/
├─ icons/
├─ screens/
└─ ui/
```

## 시작 화면 수정

- 기존 시작 화면 이미지에서 하단의 구운 `이 기기에서 로그인 유지` 원형 체크와 글씨 영역을 제거했다.
- 새 파일: `public/assets/v12/screens/start_screen_clean_v810.webp`
- 실제 버튼은 DOM 버튼으로 복구했다.
- 버튼 클래스: `hit-keep v810-keep-button`
- 기본 상태는 민트 버튼, 체크 상태는 골드 버튼을 사용한다.
- 버튼 내부에는 실제 원형 체크 인디케이터와 텍스트가 표시된다.

## UI 화면 수정

장비 / 상점 / 미션 / 도감 / 마을 화면은 `v810-screen` 클래스를 추가해 다음 스타일을 적용한다.

- 720x1280 세로형 배경 WebP 적용
- 카드 최소 높이 증가
- 카드 내부 아이콘 크기와 텍스트 영역 분리
- 버튼은 구운 글씨가 없는 blank 버튼 에셋 사용
- 하단 fixed 네비게이션 위로 콘텐츠가 가려지지 않도록 안전 패딩 추가

## 낚시 화면 수정

- PixiJS 플레이어/찌/물고기 에셋을 v12 렌더드 PNG 기준으로 교체
- v12 물고기 크기 기준에 맞춰 잡힘 연출 스케일을 `/720` 기준으로 재조정
- 지역 배경은 v12 세로형 WebP 배경으로 연결

## 검증

아래 명령으로 통과 확인했다.

```bash
npm run validate
npm run typecheck
npm run build
```

검증 스크립트는 `tools/check-v810-asset-fit-start-keep.mjs`에 추가했다.

## 주의

이번 패치는 완전한 최종 게임 UI가 아니라, 사용자가 제공한 렌더드 에셋을 실제 화면 구조에 맞게 다시 붙이고 겹침을 제거하는 대형 복구 패치다. 다음 단계에서는 화면별 전용 9-slice 패널화, PixiJS atlas 최적화, 레이어드 2.5D 애니메이션을 추가하는 것이 좋다.
