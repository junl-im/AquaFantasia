# AquaFantasia Patch 03 - v1.4 Visual Boost

## 핵심 변경

- 전체 UI를 프리미엄 글래스/오로라/수중 레이어 기반으로 보정했습니다.
- 낚시 화면에 광원, 물결, 버블, 물고기 그림자 드리프트 효과를 추가했습니다.
- 마을 지역 카드를 최신 모바일 카드형 인터페이스로 정리했습니다.
- 경험치 진행 바를 캐릭터 아래에 추가했습니다.
- 포획 결과와 업적 달성을 브라우저 alert 대신 토스트 알림으로 일부 전환했습니다.
- 모바일 진동 피드백을 추가했습니다. 지원하지 않는 기기에서는 자동 무시됩니다.
- 상단에 그래픽 고급/절전 토글을 추가했습니다.
- 사용자의 `prefers-reduced-motion` 설정을 감지해 애니메이션을 자동 축소합니다.
- 긴 스크롤 화면에 `content-visibility: auto`를 적용해 렌더링 부담을 줄였습니다.
- 낚시 타이머를 추적/정리해 화면 이동 시 남은 타이머가 UI를 건드리는 문제를 줄였습니다.
