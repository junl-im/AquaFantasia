# AquaFantasia Patch 02 - v1.3 Fish Data

## 핵심 변경

- 어종 데이터를 `data/fish.json`으로 분리했습니다.
- 6개 지역별 출현 어종과 희귀도 범위를 적용했습니다.
- 확률은 `weight` 기반 가중치 추첨 방식입니다.
- 물고기마다 가격 범위, 경험치, 설명, 이모지를 갖습니다.
- 도감 화면에서 발견 진행도와 어종 목록을 확인할 수 있습니다.
- 가방과 기록 화면에서 희귀도 배지를 표시합니다.

## 저장 데이터 호환

- 새 저장 키: `aqua_v1.3`
- 기존 키 `aqua_v1.2`, `aqua_v1.1`도 계속 읽습니다.
- 기존 유저 데이터는 초기화하지 않아도 됩니다.
