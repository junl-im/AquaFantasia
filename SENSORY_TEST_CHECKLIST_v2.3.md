# Sensory Test Checklist v2.3

1. 마을 화면에서 `SENSORY ENGINE 2.3` 패널이 보이는지 확인
2. 패널을 터치했을 때 `LIVE` 배지가 표시되는지 확인
3. 사운드 ON/OFF 버튼이 동작하는지 확인
4. 햅틱 ON/OFF 버튼이 동작하는지 확인
5. 타격감 버튼을 눌렀을 때 0.6x, 1x, 1.35x, 1.5x로 순환하는지 확인
6. 테스트 버튼에서 짧은 사운드와 임팩트 연출이 나오는지 확인
7. 낚싯대 던지기, 입질, 릴 감기, 포획 성공, 실패 사운드 확인
8. 보스 조우 시 저음 경고음과 화면 임팩트 확인
9. 새로고침 후 감각 설정이 유지되는지 확인
10. GitHub Actions `npm run validate`가 통과하는지 확인
