# Aqua Fantasia v6.6.0 Clean Replace Guide

이번 버전은 레퍼런스 아트 리마스터 패치입니다. 이전 누적 통파일, 과거 패치 문서, 오래된 런타임이 남으면 화면에 옛 UI가 섞일 수 있으므로 전체 교체를 권장합니다.

## 권장 적용 순서

1. 저장소 폴더에서 `.git` 폴더만 남깁니다.
2. 나머지 기존 파일을 삭제합니다.
3. `AquaFantasia_v6.6.0_REFERENCE_ART_REMASTER_FULL_PROJECT.zip`을 저장소 루트에 압축 해제합니다.
4. GitHub Desktop에서 커밋/푸시합니다.
5. 모바일 PWA는 기존 아이콘을 삭제 후 다시 설치합니다.

## 핵심 변경

- 텍스트가 박힌 `START` 버튼 이미지를 텍스트 없는 2.5D 버튼으로 교체
- 레퍼런스 이미지에서 플레이어, 찌, 물고기, 게이지, 도감 슬롯, 아이콘류를 크롭/정리하여 반영
- 로그인 바다 배경을 텍스트 없는 세로형 2.5D 낚시 배경으로 재구성
- 첫 화면과 마을 화면의 기술용 문구를 게임용 문구로 정리
- 세로 전용 방향 잠금 유지
