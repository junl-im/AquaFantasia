# AquaFantasia v2.0.100

## v2.0.100 변경사항

이번 패치는 새 콘텐츠 추가가 아니라 **UI 안정화 루트 복구 패치**입니다. v2.0.98에서 복구한 하단 메뉴, 마을 HUD, 우측 조작바, 개척 카드, 낚시 화면 복구 CSS가 앱 버전 변경 뒤에도 계속 적용되도록 `v2100-ui-stability-root` 기준을 추가했습니다.

- `APP_VERSION`이 `2.0.100`으로 올라가도 v2.0.98 UI 복구 레이어가 비활성화되지 않도록 버전 고정 CSS를 안정화 루트 클래스로 재적용했습니다.
- 시작 화면의 `이 기기에 로그인 유지` 체크 영역 대비, 체크 표시 크기, 글씨 가독성을 보강했습니다.
- 마을 HUD를 짧고 얇은 카드 형태로 유지하고 `Lv.10+` 표기 칸이 잘리지 않도록 레벨 영역을 보정했습니다.
- 우측 상단 조작 레일의 클릭 우선순위를 보강해 건설/상점/출항/확대/축소/원점 버튼이 월드 터치와 충돌하지 않도록 했습니다.
- 하단 메뉴는 우측 아래 한줄형 `홈 / 가방 / 퀘스트 / 지도`만 살아 있도록 구형 도크 숨김 가드를 추가했습니다.
- 개척 패널, 캐릭터/건물/건설 카드, 메뉴 닫기 X 버튼이 우측 상단 아쿠아 버튼 기준을 유지하도록 보강했습니다.
- 상점/가방/퀘스트/지도 메뉴는 `runtime-content`, `v2097-menu-content`, `v2098-menu-content` 스크롤 루트 구조를 유지합니다.
- `tools/check-v2100-ui-stability.mjs`를 추가하고 `npm run validate`에 연결해 버전 동기화, 안정화 루트, 하단 메뉴 기준, 패키지 청결, registry 오염 문자열을 함께 검사합니다.

## v2.0.98-v2.0.99 유지 내용

- 하단 메뉴바는 우측 아래 한줄형 `홈 / 가방 / 퀘스트 / 지도` 구조를 유지합니다.
- 우측 상단 조작 레일과 HUD 공간 조정, 건설 버튼 상태 동기화, 낚시 화면 복구, 상점 본문 호환 구조, 마을 터치 기준 보정은 보존했습니다.
- v2.0.99의 패키지 청결 가드, 백업/빌드 산출물 제거, 내부 registry 오염 문자열 검사는 유지했습니다.
- 캐릭터 방향 로직은 수정하지 않았습니다.

## 검증

- `npm run validate` 통과
- 금지 registry 문자열 없음 확인
- 루트 Markdown은 `README.md` 하나만 유지
- 금지 산출물(`node_modules`, `dist`, `reports`, 백업 폴더, `.log`, `*_NOTES.md`) 미포함 확인
- `AquaFantasia_v2_0_100_full.zip` 새 압축 해제 후 `npm run validate` 통과
- `AquaFantasia_v2_0_99_full.zip`에 `AquaFantasia_v2_0_100_patch.zip` 덮어쓰기 후 `npm run validate` 통과
