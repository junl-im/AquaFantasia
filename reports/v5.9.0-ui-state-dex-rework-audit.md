# v5.9.0 UI State Router + 2.5D Fish Dex Rework Audit

## 문제
누적 패치 과정에서 v5.3 내비게이터, v5.4 상점 FAB, v5.5 터치 코치, 디버그 배지가 서로 다른 시점에 `document.body`에 붙으면서 로그인/마을/도감 화면에도 낚시 전용 조작 UI가 노출되었습니다.

## 조치
- `src/runtime/v59-ui-state-dex-rework.js`를 추가해 화면 상태를 감지하고 화면별 UI 허용 정책을 적용했습니다.
- `.aqua-nav-v53`, `.aqua-shop-fab-v54`, v52~v55 개발용 패널을 플레이어 화면에서 숨겼습니다.
- 낚시 화면이 아닐 때 `.aqua-fishing-v53-ui`, `#aqua-v55-touch-coach`, `.aqua-v55-meter`, `#aqua-v49-action-layer`가 노출되지 않도록 했습니다.
- 도감/가방 카드에 v5.9 2.5D 물고기 포트레이트와 카드 스킨을 적용했습니다.

## 검증 포인트
- 로그인 첫 화면에 `던지기`, `챔질`, `상점` 고정 버튼이 보이지 않아야 합니다.
- 낚시 화면 진입 후에는 기존 캐스팅/릴링 로직이 유지되어야 합니다.
- 도감 화면은 지역별 2.5D 물고기 카드로 재스킨되어야 합니다.
