# AquaFantasia v2.1.7

## v2.1.7 변경사항

- 버전을 `2.1.7`로 올리고 `package.json`, `package-lock.json`, `APP_VERSION`, service worker cache, offline badge/version, `README.md`를 동기화했습니다.
- `v217-aqua-ui-touch-fishing-normalize-root`를 추가해 로그인 토글, 공통 아쿠아 모달, 메뉴 페이지, 하단 메뉴바, 상점/출항 터치, 낚시 화면을 한 번 더 정상화했습니다.
- `이 기기에 로그인 유지` 버튼을 이미지 배경 없는 얇은 아쿠아 토글로 변경했습니다. 체크 상태는 `✓` 표시와 원형 체크 인디케이터로 확실히 구분됩니다.
- 캐릭터 정보창을 기준 아쿠아 카드 스타일로 복원했습니다. 캐릭터 카드 자체는 밝은 아쿠아 카드, 카드 외부 딤/블러, 우측 상단 원형 X, 안정된 스크롤 구조를 유지합니다.
- 개척 창은 모달로 열릴 때 HUD, 조작바, 조이스틱, 하단 메뉴바, 마을 알림이 보이지 않도록 분리했습니다. 창 바깥 영역에는 딤/블러가 적용됩니다.
- 모든 메뉴 페이지는 더 이상 HUD를 따라다니지 않도록 했고, 해당 페이지 내용만 중앙 아쿠아 카드 안에서 열리도록 정리했습니다.
- 상점 버튼과 출항 버튼의 이벤트 라우팅을 보강했습니다. 우측 상단 조작바에서 상점/출항을 눌렀을 때 모달 잔여 상태를 정리하고 바로 해당 화면으로 이동합니다.
- 우측 하단 `홈 / 가방 / 퀘스트 / 지도` 메뉴바 크기를 줄이고 우측 하단 고정 잠금을 강화했습니다.
- 낚시게임은 핵심 화면으로 별도 보강했습니다. 출항 진입, 캐스팅 버튼, 장비바, 최근 포획, 릴 콘솔, 결과창의 포인터 영역을 분리하고 UI 겹침을 줄였습니다.
- 토스트 팝업은 계속 비활성 상태입니다.
- 캐릭터 방향 로직은 수정하지 않았습니다.

## 검증 기준

- 루트 Markdown은 `README.md` 하나만 유지합니다.
- v2.1.7 통파일 ZIP을 새로 풀어도 `npm run validate`가 통과해야 합니다.
- v2.1.6 통파일에 v2.1.7 패치 ZIP을 그대로 덮어쓴 뒤 `npm run validate`가 통과해야 합니다.
- ZIP에는 `node_modules`, `dist`, `reports`, `AquaFantasia_backup_v1`, `.log`, `*_NOTES.md`를 포함하지 않습니다.
- `packages.applied-caas`, `applied-caas-gateway`, `10.192.`, `internal.api.openai` 문자열이 `package.json`/`package-lock.json`에 없어야 합니다.
