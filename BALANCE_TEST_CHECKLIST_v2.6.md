# Balance Test Checklist v2.6

1. 상단 버전이 v2.6으로 보이는지 확인
2. 마을에 Balance Coach 카드가 표시되는지 확인
3. 지역 카드에 추천/준비도 미터가 표시되는지 확인
4. 전투력이 낮은 상태에서 용궁/차원의 바다 선택 시 경고 토스트가 뜨는지 확인
5. 호수/강에서 초반 10마리 포획이 너무 어렵지 않은지 확인
6. 물고기 판매 후 추천 행동이 강화/장비 구매로 바뀌는지 확인
7. 낚싯대와 미끼 가격이 이전보다 부담이 줄었는지 확인
8. 강화 비용과 성공률이 장비 화면에 정상 표시되는지 확인
9. 새로고침 후 aqua_v2.6 저장 데이터가 유지되는지 확인
10. GitHub Actions에서 npm run validate가 성공하는지 확인
