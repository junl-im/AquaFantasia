# AquaFantasia v2.1.54

## v2.1.54 변경사항

- 우측 상단 메뉴바를 아이콘 중심의 초소형 24px 셀로 다시 정리했습니다.
- 뒤쪽 큰 테이블/배경 프레임은 제거하고, 아이콘 주변의 얇은 반투명 1중 테두리만 유지했습니다.
- 버튼 간격을 넓히고 텍스트 라벨은 시각적으로 숨겨 상단 HUD와 겹치는 느낌을 줄였습니다.
- 낚시 릴링 실전 화면에서 포획/텐션/저항 게이지가 조작 버튼 위에 항상 보이도록 v2.1.54 전용 고정 게이지 레이어를 추가했습니다.
- 낚시 준비 UI와 실전 UI를 더 강하게 분리해 릴링 중 게이지가 다른 카드 뒤에 가려지거나 왼쪽으로 밀리는 문제를 줄였습니다.
- 감기/풀기 버튼은 하단 고정 콘솔로 유지하되, 보조 터치존과 구형 카드 조각은 실전 단계에서 숨김 처리합니다.
- 메뉴/상점/개척/하단 아이콘 클리핑 안정화 정책은 유지합니다.
- 타일 픽셀 크기는 유지합니다. v2.1.54는 다이아몬드 터치 점수만 `0.934`로 소폭 조정하며, 타일 픽셀 축소는 세이브 좌표, 건물 footprint, NPC 이동, 충돌 판정, 카메라 경계 마이그레이션 전까지 보류합니다.
- 루트 버전 파일 `APP_VERSION`은 만들지 않습니다. 버전 기록은 README와 코드 상수로만 관리합니다.

## 검증

- `npm run validate`
- full ZIP / patch ZIP 무결성 확인
- full ZIP 새 압축 해제 후 validate
- 이전 full ZIP에 patch ZIP 덮어쓰기 후 validate

## 배포 메모

- `node_modules`, `dist`, `reports`, `backup`, `logs`, `_NOTES`, `APP_VERSION`은 배포 ZIP에 포함하지 않습니다.
- GitHub Actions에서는 `npm ci` 후 생성되는 `node_modules`가 있어도 validate가 실패하지 않아야 합니다.
