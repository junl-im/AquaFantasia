# AquaFantasia v2.1.56

## v2.1.56 변경사항

- 우측 상단 마을 메뉴바가 기존 CSS에 덮여 변화가 없어 보이던 문제를 실제 렌더링 DOM에 직접 적용하는 inline-important 하드락으로 수정했습니다.
- 마을 조작 메뉴는 3x2 배열, 24px 아이콘 셀, 13px 가로 간격, 투명 배경, 1중 반투명 테두리만 유지합니다.
- 낚시 릴링 중 포획/텐션/저항 게이지가 버튼 위에서 항상 보이도록 v2.1.56 전용 실전 게이지 레이어를 추가했습니다.
- 감기/풀기 버튼과 실전 게이지가 서로 가리지 않도록 준비 UI와 릴링 UI를 더 분리했습니다.
- 타일 픽셀 크기는 유지합니다. v2.1.56은 다이아몬드 터치 점수만 `0.930`으로 소폭 조정하며, 타일 픽셀 축소는 세이브 좌표, 건물 footprint, NPC 이동, 충돌 판정, 카메라 경계 마이그레이션 전까지 보류합니다.

## 설치/검증

```bash
npm ci
npm run validate
npm run build
```
