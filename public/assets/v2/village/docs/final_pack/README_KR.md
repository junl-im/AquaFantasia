## 사용 팁
- 현재 이미지는 시트 형태입니다. 실제 게임에 넣을 때는 필요한 오브젝트 단위로 잘라 사용하세요.
- 80x80 타일맵과 섞어 쓸 때는 큰 오브젝트는 2x2~4x4 타일 기준으로 배치하는 것을 추천합니다.
- NPC 순찰은 8방향 스프라이트를 기준으로 이동 애니메이션 프레임을 추가 제작하면 가장 자연스럽습니다.
- 벤치 앉기 포즈는 벤치 오브젝트와 레이어를 분리해 배치하면 좋습니다.
