# AquaFantasia v2.1.0

## v2.1.0 변경사항

이번 패치는 버전 흐름을 `2.1.x` 계열로 전환하고, 새 콘텐츠를 추가하지 않은 **아쿠아 카드 UI/UX 기초 재정렬 패치**입니다. 캐릭터 정보창의 밝은 아쿠아 카드, 카드 바깥 딤/블러, 우측 상단 원형 닫기 버튼 기준을 전체 메뉴와 마을 조작 UI에 맞추는 것을 우선했습니다.

- 버전 기준을 `2.1.0`으로 올리고 이후 패치가 `2.1.1`, `2.1.2` 순서로 이어질 수 있도록 `validate-clean`의 허용 버전 규칙을 `2.0.x`와 `2.1.x`로 확장했습니다.
- `v210-aqua-ui-foundation-root`를 추가해 `v2100-ui-stability-root`에 의존하던 복구 CSS 위에 2.1.x 전용 UI 기준을 한 번 더 고정했습니다.
- 마을 첫 화면 배치를 재정렬했습니다.
  - 왼쪽 위: 짧은 아쿠아 HUD
  - HUD 아래: 개척 바
  - 오른쪽 위: 건설/상점/출항/확대/축소/원점 조작바
  - 왼쪽 아래: 조이스틱
  - 오른쪽 아래: `홈 / 가방 / 퀘스트 / 지도` 한줄 메뉴바
- 하단 메뉴바 폭을 줄여 왼쪽 아래 조이스틱과 겹치지 않도록 하고, 구형 도크와 버튼 스프라이트 잔상이 다시 보이지 않도록 버튼 배경/가상 요소를 차단했습니다.
- 캐릭터 정보창, 개척창, 건물 정보창, 건설창, 일반 메뉴창을 같은 아쿠아 카드 톤으로 통일하고 카드 바깥 흐림/딤 효과를 보강했습니다.
- 일반 메뉴 화면의 본문을 중앙 아쿠아 카드형 스크롤 루트로 정리해 가방/퀘스트/지도/상점 내용이 같은 구조로 보이게 했습니다.
- `tools/check-v210-aqua-ui-foundation.mjs`를 추가하고 `npm run validate`에 연결해 버전 동기화, 2.1.x UI 루트, HUD/개척바/조작바/조이스틱/하단 메뉴바 기준, 패키지 청결, registry 오염 문자열을 검사합니다.

## 유지 내용

- v2.0.98에서 복구한 낚시 화면, 하단 메뉴 4개 구조, 상점 본문 호환 구조, 건설 버튼 상태 동기화는 유지했습니다.
- v2.0.99의 패키지 청결 가드, 백업/빌드 산출물 제거, 내부 registry 오염 문자열 검사는 유지했습니다.
- v2.0.100의 버전 독립 UI 안정화 루트는 하위 호환 보호막으로 유지했습니다.
- 캐릭터 방향 로직은 수정하지 않았습니다.

## 검증

- `npm run validate` 통과 필요
- 금지 registry 문자열 없음 유지
- 루트 Markdown은 `README.md` 하나만 유지
- 금지 산출물(`node_modules`, `dist`, `reports`, 백업 폴더, `.log`, `*_NOTES.md`) 미포함 유지
- 통파일 ZIP과 v2.0.100 기준 덮어쓰기 패치 ZIP 둘 다 제공

## 패키징 규칙

- 루트 Markdown 파일은 `README.md` 하나만 유지합니다.
- `node_modules`, `dist`, `reports`, 백업 폴더, 임시 로그, `*_NOTES.md`는 ZIP에 포함하지 않습니다.
- 내부 registry 오염 문자열 검사를 유지합니다.
