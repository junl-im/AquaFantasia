# AquaFantasia v2.1.52

## v2.1.52 변경사항

- 우측 상단 마을 메뉴바를 핫픽스했습니다.
  - 뒤쪽 큰 배경/테이블 프레임은 계속 제거합니다.
  - 아이콘 버튼은 31px급 반투명 1중 셀로 줄였습니다.
  - 3x2 배열, 간격 8px 기준으로 다시 잡아 버튼끼리 겹치지 않게 조정했습니다.
- 낚시 실전 화면을 우선 수정했습니다.
  - 입질 후 릴링 단계에서 포획/텐션/저항 게이지가 화면 상단 고정 카드로 항상 보이게 했습니다.
  - 감기/풀기 버튼은 하단 고정 콘솔로 정리했습니다.
  - 보조 터치존은 실전 게이지와 버튼을 가리지 않도록 숨겼습니다.
  - 릴 패널은 게이지 아래쪽 보조 정보 영역으로 밀어 UI 겹침을 줄였습니다.
- 하단 메뉴/상점/개척/페이지 카드 정리 정책은 v2.1.51 기준을 유지합니다.
- 타일 픽셀 크기는 유지합니다. v2.1.52는 다이아몬드 터치 점수만 `0.938`로 소폭 조정하며, 타일 픽셀 축소는 세이브 좌표, 건물 footprint, NPC 이동, 충돌 판정, 카메라 경계 마이그레이션 전까지 보류합니다.
- `node_modules`는 CI 작업환경에서는 허용하지만, full/patch ZIP에는 포함하지 않습니다.

## 실행

```bash
npm ci
npm run validate
npm run build
```

## 패키징 원칙

- 루트 Markdown은 `README.md`만 유지합니다.
- `APP_VERSION`, `node_modules`, `dist`, `reports`, `backup`, `logs`, `_NOTES`, `*.log`는 배포 ZIP에 포함하지 않습니다.
