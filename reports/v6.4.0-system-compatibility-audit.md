# v6.4.0 System Compatibility Audit

## 점검 항목

- 시작 화면에 낚시 HUD, 상점 FAB, 디버그 배지가 노출되지 않도록 유지
- 기기 시작 방향을 기준으로 레이아웃 보정
- requestFullscreen 및 screen.orientation.lock은 사용자 제스처 후 조용히 시도
- GitHub Actions는 Node 24를 유지하고 npm cache lockfile 의존을 제거
- Service Worker는 v6.4 캐시만 유지하고 이전 aqua-fantasia 캐시를 정리
- HTML 요청은 네트워크 우선으로 처리하여 오래된 index.html 잔존 위험 완화
- 도감/상점/장비/미션/낚시의 저장 데이터가 서로 연결되도록 조정

## 주요 변경

- 얼음 낚시터, 폭풍 외해 추가
- 낚싯줄 강화 추가
- 미션 보상 수령과 수역 해금 연결
- 낚시 결과 카드 추가
- 도감 물고기 확장
- 안전지대 위치/크기를 지역 난이도, 장비, 보스 등급에 따라 조절

## 확인 결과

정적 검증, 런타임 토큰 검증, 자산 존재 검증, Atlas 메타데이터 검증을 통과했습니다.
