# Aqua Fantasia v6.9.0 Clean Replace Guide

## 권장 적용 방식
기존 누적 파일이 남으면 예전 통파일/런타임/캐시가 섞일 수 있으므로, 가능하면 전체 교체를 권장합니다.

1. GitHub Desktop에서 저장소 폴더 열기
2. `.git` 폴더만 남기고 나머지 파일 삭제
3. `AquaFantasia_v6.9.0_START_BACK_BITE_UI_FULL_PROJECT.zip` 압축 해제
4. GitHub Desktop에서 커밋/푸시
5. 카카오톡 인앱 브라우저는 링크 창을 완전히 닫고 다시 열기

## 덮어쓰기 패치 사용 조건
v6.8.0을 정확히 클린 적용한 상태라면 덮어쓰기 패치도 사용할 수 있습니다.

## 주요 확인 포인트
- 시작 화면 상단 공지사항/고객센터/설정이 보이지 않는지
- 시작 화면 하단 가방/도감/업적/이벤트/상점이 보이지 않는지
- 로그인 유지 체크 상태가 구분되는지
- 입질 시 화면 중앙에 당기기 안내가 뜨는지
- 낚시터에서 뒤로가기 시 마을 복귀 확인창이 뜨는지
- 마을에서 뒤로가기 시 종료 확인창이 뜨는지
