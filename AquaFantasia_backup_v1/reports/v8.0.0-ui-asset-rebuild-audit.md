# Aqua Fantasia v8.0.0 UI Asset Rebuild Audit

## 목적
v7.6.0에서 장비/상점/미션 화면의 가독성 저하와 렌더드 게임풍 부족 문제가 확인되어, 사용자 제공 PNG 시트를 실제 런타임 에셋으로 다시 반영했다.

## 반영
- 낚시.zip의 렌더드 배경 시트에서 Ocean/Lake/River/Stream/DeepSea/Town Waterfront WebP 배경을 재생성했다.
- 렌더드 물고기 시트에서 fish_01~fish_29 PNG를 추출해 기존 벡터풍 물고기를 대체했다.
- UI/아이콘 시트에서 낚싯대, 릴, 낚싯줄, 미끼, 찌, 코인, 가방, 상점, 하단 nav 아이콘을 추출했다.
- 장비/상점/미션/도감 화면을 세로 스크롤 카드 구조로 강제 재정렬했다.
- 첫 시작 화면의 `익명 서버연동`과 `이 기기에서 로그인 유지`가 겹쳐 보이던 중복 텍스트/배경 레이어를 제거하고, 로그인 유지 영역은 투명 hitbox + 체크 표시만 남겼다.
- PixiJS 낚시 화면의 플레이어/물고기/찌 스케일을 새 렌더드 PNG 크기에 맞게 재조정했다.

## 검증 포인트
- `npm run validate`
- `npm run typecheck`
- `npm run build`
- 모바일 세로 화면에서 장비/상점/미션 진입 후 하단 nav 위까지 스크롤 가능해야 한다.
- 하단 nav는 화면 드래그와 관계없이 최하단 고정 상태를 유지해야 한다.
