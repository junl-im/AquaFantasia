# v8.4.0 덮어쓰기 적용 가이드

1. 기존 AquaFantasia 프로젝트 루트에 ZIP 내용을 그대로 덮어쓴다.
2. 터미널에서 아래 명령을 실행한다.

```bash
npm install
npm run typecheck
npm run validate
npm run build
```

3. GitHub Desktop에서 변경사항을 commit/push 한다.
4. GitHub Pages 배포 후 모바일 브라우저에서 아래를 확인한다.

## 확인 체크리스트
- 낚시 화면이 v13 카드 구성도 안에 작게 갇히지 않고 전체 화면 배경으로 보이는지.
- 낚시 화면에서 낚싯대/미끼/오늘 목표 카드가 사라졌는지.
- `낚시 시작` 버튼이 유지되고 충분히 큰지.
- 최근 포획 패널이 하단 메뉴 바로 위에 있는지.
- 하단 네비가 이상한 배경이 아니라 실제 아이콘 버튼형으로 보이는지.
- 랭킹에 가짜 유저 여러 명이 보이지 않는지.
- 카카오 브라우저에서 뒤로가기/메뉴 이동 후 다시 액션하면 세로 몰입 화면이 재정렬되는지.
- 회전 경고/가로 레이아웃이 새로 생기지 않았는지.
