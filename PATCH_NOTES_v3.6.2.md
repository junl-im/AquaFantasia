# AquaFantasia Patch 25.2 - v3.6.2 Performance Core Hotfix

이번 패치는 콘텐츠 추가보다 모바일 랙 감소와 초기 로딩 안정화에 집중했습니다.

## 핵심 수정

- 모바일/카카오/저메모리/데이터 절약 환경은 기본 `perf-lite` 모드로 시작합니다.
- Service Worker 설치 시 모든 SVG 에셋을 한 번에 캐시하지 않고, 핵심 파일만 선캐시합니다.
- 나머지 에셋은 화면에서 필요할 때 런타임 캐시됩니다.
- 저장 시 과거 모든 버전 키에 반복 저장하던 구조를 제거했습니다.
- 현재 버전, 직전 핫픽스, `aqua_latest_state`만 저장해 localStorage jank를 줄였습니다.
- v31~v36 에셋 워밍은 Ultra/고성능 환경에서만 제한적으로 실행됩니다.
- 모바일 저사양에서는 backdrop blur, 오래된 대형 패널 애니메이션, 과도한 물고기 그림자 수를 줄입니다.
- 시작 직후 FPS 점검을 늦춰 첫 화면 초기화와 겹치지 않게 했습니다.

## 기대 효과

- 첫 접속/업데이트 직후 캐시 설치 부담 감소
- 낚시 화면 진입 시 순간 끊김 감소
- 판매/저장/클라우드 저장 타이밍에서 멈칫함 감소
- 카카오 인앱브라우저 및 저사양 Android에서 더 안정적인 프레임 유지
