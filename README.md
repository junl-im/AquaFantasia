# AquaFantasia v2.1.29

## v2.1.29 변경사항

- 사용자가 새로 제공한 `player.zip`의 파일명 기준 8방향 플레이어 프레임을 그대로 적용했습니다.
  - 적용 경로: `public/assets/v2129/characters/player`
  - 8방향 × 4프레임 = 32개 PNG
  - 이제 `east`는 `player_east_*`, `west`는 `player_west_*`를 직접 사용합니다.
  - v2.1.28의 좌우 반전 보정 프레임과 코드 보정에 의존하지 않습니다.
- 플레이어 방향 매핑을 전부 identity로 고정했습니다.
  - east → east
  - west → west
  - northeast → northeast
  - northwest → northwest
  - 나머지 방향도 파일명과 동일하게 매핑합니다.
- 걷기 모션은 너무 빠르게 떨리지 않도록 프레임 전환, 좌우 흔들림, 스쿼시 값을 낮췄습니다.
- 우측 상단 조작바는 투명 테두리와 간격을 유지하며 아이콘 겹침을 줄였습니다.
- 우측 하단 메뉴바는 마을 화면에서도 메뉴 화면과 비슷하게 보이도록 아이콘과 버튼 영역을 조금 키웠습니다.
- 가방, 퀘스트, 지도, 상점, 건설, 건물 상세, 캐릭터 정보, 개척, 종료 팝업은 공통 아쿠아 카드 톤과 글씨 번짐 방지 규칙을 유지합니다.
- 낚시 화면은 릴 패널, 캐스팅 버튼, 터치존 중앙 고정과 대형 아이콘 제한을 유지합니다.

## 검증

- `npm run validate`는 `tools/check-v2129-player-filename-direction-lock.mjs`를 실행합니다.
- 검증 항목:
  - v2.1.29 버전 동기화
  - CSS 괄호/중괄호/대괄호 균형
  - v2129 플레이어 32프레임 존재
  - 플레이어 모션 경로가 v2129를 사용하는지
  - 모든 플레이어 방향 매핑이 파일명과 동일한지
  - 조이스틱 v2129 transform 변수
  - 우측 상단/하단 메뉴 v2129 클래스
  - 낚시 v2129 안정화 클래스
  - 로그인 유지 토글 보호 규칙
