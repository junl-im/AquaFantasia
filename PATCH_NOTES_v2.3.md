# Aqua Fantasia Patch Notes v2.3 — Sensory Visual Max

## 핵심 변경
- Web Audio API 기반 사운드 엔진 추가
- 캐스팅, 입질, 릴 감기, 퍼펙트 카운터, 성공/실패, 강화, 보상 사운드 추가
- 햅틱 설정, 볼륨 설정, 타격감 강도 설정 추가
- 마을 화면에 `SENSORY ENGINE 2.3` 패널 추가
- 터치 리플, 히트스톱, 임팩트 플래시, 릴 스트레스 연출 추가
- Save-Data/저사양 환경에서는 자동 절전 그래픽 우선 적용
- `aqua_v2.3` 저장 키 추가, v2.2 이하 데이터 호환 유지
- Service Worker 캐시 `v2.3.0` 갱신

## 운영 메모
- 모바일 브라우저 정책상 오디오는 첫 사용자 터치 이후 활성화됩니다.
- 진동은 브라우저와 기기 지원 여부에 따라 자동 무시될 수 있습니다.
- 외부 음원 파일을 쓰지 않고 브라우저에서 직접 합성하므로 GitHub Pages 무료 배포에 적합합니다.
