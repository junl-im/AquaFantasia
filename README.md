# AquaFantasia v2.1.49

## v2.1.49 변경사항

- UI 겹침/배치/배정 문제를 계속 훑고, 아이콘·메뉴 프레임·상점·개척·낚시 화면을 다시 정리했습니다.
- 우측 상단 마을 메뉴는 뒤쪽 배경 프레임을 계속 제거하고, 아이콘을 감싸는 반투명 1중 셀만 비례 크기로 유지합니다.
- 하단 홈/가방/퀘스트/지도 아이콘은 투명 여백을 더 확보한 128x128 PNG로 다시 정리하고, 버튼 내부 클리핑을 강화했습니다.
- 가방/퀘스트/지도/상점 페이지의 중복 제목 잔상은 `data-v2149-page-title-clean` 가드로 계속 차단합니다.
- 상점 아이템 카드는 글 영역과 골드 구매 버튼 영역을 분리하고, 긴 설명이 가격 배지를 덮지 않도록 더 좁은 화면 기준으로 보강했습니다.
- 개척 후보지/항로 조사 영역은 카드/칩/버튼 간격을 다시 맞춰 덩그러니 글만 떠 보이는 문제를 줄였습니다.
- 낚시 화면은 준비 단계와 릴링 단계를 더 강하게 분리하고, 릴링 중 본 게임 콕핏은 좌우 inset 고정 방식으로 화면 밖 밀림을 방지합니다.
- 낚시 바다물길 표기는 하나의 카드만 유지하며, 로드/미끼·코치·캐스팅 버튼은 겹치지 않게 압축 배치합니다.
- 타일 픽셀 크기는 유지합니다. v2.1.49는 다이아몬드 터치 점수만 `0.944`로 아주 작게 조정하며, 타일 픽셀 축소는 세이브 좌표, 건물 footprint, NPC 이동, 충돌 판정, 카메라 경계 마이그레이션 전까지 보류합니다.
- `node_modules`가 GitHub Actions의 `npm ci` 후 존재해도 validate가 실패하지 않는 정책을 유지합니다. 단, node_modules가 Git에 추적되거나 full/patch ZIP에 들어가는 것은 계속 차단합니다.
