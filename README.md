# AquaFantasia v2.1.57

## v2.1.57 변경사항

- 우측 상단 메뉴바를 기존 클래스 크기 조정 방식이 아니라 실제 마이크로 툴바 구조로 교체해 버튼이 커지거나 서로 붙는 문제를 원천적으로 줄였습니다.
- 상단 메뉴 버튼은 20px 셀, 12px 아이콘, 10px 간격의 3x2 배열로 고정하고 뒤쪽 배경/테이블 프레임과 텍스트 라벨을 제거했습니다.
- 낚시 릴링 중 포획/텐션/저항 실전 게이지를 감기/풀기 버튼 바로 위에 고정하고, 구형 준비/패널 UI가 게이지를 가리지 않도록 숨김 범위를 강화했습니다.
- 타일 픽셀 크기는 유지합니다. v2.1.57은 다이아몬드 터치 점수만 `0.928`로 소폭 조정하며, 타일 축소는 세이브 좌표/건물 footprint/NPC 이동/충돌/카메라 경계 마이그레이션 전까지 보류합니다.

# AquaFantasia v2.1.56 이전 기록

## v2.1.56 변경사항

- 우측 상단 마을 메뉴바가 기존 CSS에 덮여 변화가 없어 보이던 문제를 실제 렌더링 DOM에 직접 적용하는 inline-important 하드락으로 수정했습니다.
- 마을 조작 메뉴는 3x2 배열, 24px 아이콘 셀, 13px 가로 간격, 투명 배경, 1중 반투명 테두리만 유지합니다.
- 낚시 릴링 중 포획/텐션/저항 게이지가 버튼 위에서 항상 보이도록 v2.1.56 전용 실전 게이지 레이어를 추가했습니다.
- 감기/풀기 버튼과 실전 게이지가 서로 가리지 않도록 준비 UI와 릴링 UI를 더 분리했습니다.
- 타일 픽셀 크기는 유지합니다. v2.1.56은 다이아몬드 터치 점수만 `0.930`으로 소폭 조정하며, 타일 픽셀 축소는 세이브 좌표, 건물 footprint, NPC 이동, 충돌 판정, 카메라 경계 마이그레이션 전까지 보류합니다.
