# v5.5.2b CI File Repair Audit

## Root cause

The v5.5.2 overwrite ZIP included `tools/check-v49-action-mobile.mjs`, `index.html`, and `sw.js`, but omitted the referenced runtime module `src/runtime/v49-action-mobile-patch.js`. This caused GitHub Actions to fail during `npm run runtime:check`.

## Repair policy

The hotfix is additive: it restores the missing v4.9 action runtime and its validation support files without changing APP_VERSION, cache namespace, or v5.5.2 runtime order.

## Expected result

`npm run runtime:check` no longer stops at `[check-v49-action-mobile] missing src/runtime/v49-action-mobile-patch.js`.
