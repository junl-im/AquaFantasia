# AquaFantasia Art Bible

## Core identity

- Genre: Fantasy Ocean RPG
- Style: Stylized SD
- Camera: Quarter View
- Mood: Warm, Bright, Adventure
- Rendering: 2.5D Hand Painted
- Target: Mobile Portrait
- Resolution: HD

## Priority direction

The first village should create the feeling of a living island town before the user sees any fishing button. Prioritize town square, shops, inn, NPCs, and paths over fish collection, encyclopedia expansion, or extra UI buttons.

## First village layout

- Center: large fountain, plaza tiles, benches, flowers, flags
- Left: Fish Market
- Right: Fishing Guild
- Top: Village Inn
- Bottom: Harbor entrance

## Building consistency

Shared motifs:
- Blue roof
- White/cream wall
- Wooden terrace and beams
- Flower pots
- Seashells, coral, blue crystal accents

Building notes:
- Fisherman House: blue roof, white walls, wood terrace, flower pots
- Fish Market: sailcloth canopy, fishing nets, wooden signboard
- Village Inn: large windmill, ocean pattern, warm lighting
- Harbor Office: anchor emblem, notice board, dock office props
- Fishing Guild: more prestigious silhouette, banners, fish motif
- Aquarium: curved glass/tank windows, aqua glow
- Blacksmith: forge, anvil, seaside decorations
- Storage Warehouse: crates, barrels, rope, nets

## UI direction

Recommended:
- Blue crystal
- Coral ornaments
- Seashell decorations
- Wood
- Bright beige

Avoid:
- Gold-heavy UI
- Black-heavy UI
- Lineage-like fantasy UI
- Chinese mass-market MMORPG visual language

## Icon style

- Glossy
- Fantasy
- Soft shadow
- 3D painted
- AAA mobile game quality

Needed future icons:
Fishing, Bag, Village, Map, Encyclopedia, Quest, Mail, Friends, Guild, Shop, Pearl, Gold, Energy, Settings.
