# AquaFantasia Patch 01 - v1.2 Stability

이번 패치는 기능 확장 전에 게임이 안정적으로 저장되고 Firebase 로그인 상태를 확인하기 쉽도록 만든 안정화 패치입니다.

## 적용 내용

- 상단 버전 표시를 v1.2로 변경
- Firebase 로그인 후 로그아웃 버튼 추가
- 로컬 저장/클라우드 저장 상태 뱃지 추가
- 닉네임 입력값 정리: 공백 정리, 최대 24자 제한
- 리더보드/가방/기록 표시에서 사용자 입력값 HTML 이스케이프 처리
- 전체 판매/개별 판매 안정성 보강
- 이전 v1.1 localStorage 저장 데이터와 호환 유지
- 이메일/비밀번호 입력 후 Enter 키로 로그인 가능
- 페이지 종료 전 로컬 저장 시도
- 도감 안내 문구를 다음 패치 예고 형태로 정리

## GitHub Desktop 적용 방법

1. 이 폴더 전체를 기존 저장소 폴더에 덮어씁니다.
2. GitHub Desktop에서 변경 파일을 확인합니다.
3. 커밋 메시지 예시: `Patch 01: v1.2 stability update`
4. Push origin을 누릅니다.
5. GitHub Pages 배포가 끝난 뒤 사이트를 새로고침합니다.

## Firebase 확인 체크리스트

- 익명 로그인 버튼으로 접속되는지 확인
- 이메일 회원가입/로그인이 되는지 확인
- Google 로그인 팝업이 뜨는지 확인
- 로그인 후 상단에 `로그아웃` 버튼이 표시되는지 확인
- 낚시 성공 후 `클라우드 저장됨` 표시가 뜨는지 확인
- 경쟁 화면에서 글로벌 리더보드 제출이 되는지 확인
