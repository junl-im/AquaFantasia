# Kakao Mobile Launch Guide v2.2

## 목표
카카오톡 링크에서 게임이 열릴 때 인앱브라우저 제한을 감지하고, 사용자가 기본 브라우저/Chrome/Safari로 빠르게 옮겨 플레이할 수 있게 합니다.

## 동작
1. User-Agent에 KakaoTalk 계열 문자열이 있으면 브릿지 패널을 표시합니다.
2. Android에서는 `intent://` URL로 기본 브라우저 또는 Chrome 실행을 시도합니다.
3. iOS에서는 웹에서 Safari를 강제 실행할 수 없으므로 링크 복사와 Safari 열기 안내를 제공합니다.
4. 기본 브라우저/PWA에서 `게임모드` 버튼을 누르면 전체화면, 세로 방향, 화면 켜짐 유지, Viewport 보정이 적용됩니다.

## 권장 공유 문구
카카오 공유 메시지에는 “게임이 카카오 내부 브라우저에서 열리면 `기본 브라우저` 버튼을 눌러 플레이하세요”를 함께 넣는 것이 좋습니다.
