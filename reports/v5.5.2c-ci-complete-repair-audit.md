# Aqua Fantasia v5.5.2c CI Complete Repair Audit

## Root cause

The v5.5.2b full project contained `reports/v5.5.1-runtime-error-audit.md`, but the overwrite patch ZIP omitted it. The GitHub Actions `Runtime checks` step executes `tools/check-v551-hotfix.mjs`, which requires that report file. Because the file was absent after applying the overwrite ZIP, the CI job failed with exit code 1.

## Repair strategy

Instead of shipping another narrow delta patch, v5.5.2c rebuilds the overwrite ZIP as a complete overlay. This keeps the user workflow simple: extract the ZIP at repository root, commit, and push.

## Preserved versioning

`APP_VERSION` remains `5.5.2` because the runtime code is still the v5.5.2 hotfix line. The `2c` label identifies only the packaging/CI file repair.

## Result

The repaired tree passes:

- `npm run validate`
- `npm run runtime:check`
- `npm run renderer:check`
- `npm run audit`
- `npm run atlas:check`
- `npm run clean:report`
