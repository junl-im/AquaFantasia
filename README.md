# AquaFantasia v2.1.53

## v2.1.53 변경사항

- 우측 상단 마을 메뉴바를 다시 줄였습니다.
  - 버튼은 27px급 반투명 1중 셀로 축소했습니다.
  - 3x2 배열은 유지하되 간격을 넓혀 버튼끼리 붙어 보이는 문제를 줄였습니다.
  - 뒤쪽 큰 배경/테이블 프레임은 계속 제거합니다.
- 낚시 실전 화면에 전용 배틀 게이지 HUD를 추가했습니다.
  - 릴링 중에는 포획/텐션/저항 게이지가 별도 fixed 스트립으로 항상 표시됩니다.
  - 감기/풀기 버튼은 하단 고정 콘솔로 유지하고, 보조 터치존은 계속 숨깁니다.
  - 준비 UI와 실전 UI를 더 분리해 게이지가 카드 뒤에 가려지지 않게 했습니다.
- 하단 메뉴/상점/개척/페이지 카드 정리 정책은 v2.1.52 기준을 유지합니다.
- 타일 픽셀 크기는 유지합니다. v2.1.53은 다이아몬드 터치 점수만 `0.936`으로 소폭 조정하며, 타일 픽셀 축소는 세이브 좌표, 건물 footprint, NPC 이동, 충돌 판정, 카메라 경계 마이그레이션 전까지 보류합니다.
- `node_modules`는 CI 작업환경에서는 허용하지만, full/patch ZIP에는 포함하지 않습니다.

## 실행

```bash
npm ci
npm run validate
npm run build
```

## 패키징 원칙

- 루트 Markdown은 `README.md`만 유지합니다.
- `APP_VERSION`, `node_modules`, `dist`, `reports`, `backup`, `logs`, `_NOTES`, `*.log`는 배포 ZIP에 포함하지 않습니다.
