# AquaFantasia v3.6.4 Start Flow & Immersive UX Hotfix

## 핵심 수정
- 기본 시작 데이터를 레벨 1 / 골드 0 / 닉네임 낚시꾼으로 변경했습니다.
- v3.6 이전 데모 시드 데이터(여우, Lv.12, 1,240G, 포획 0)를 자동 감지해 신규 시작 데이터로 보정합니다.
- Firebase 인증 지속성을 브라우저 세션 기준으로 낮추고, 명시적으로 시작하지 않은 이전 로그인은 자동 진입하지 않도록 조정했습니다.
- 첫 화면에서 상단 HUD/하단 메뉴가 먼저 보이지 않도록 로그인 모드를 분리했습니다.
- 몰입모드 플로팅 버튼과 수동 전체화면 배너를 숨기고, 첫 터치 순간 자동 전체화면/게임모드를 시도합니다.
- 마을 상단에 첫 플레이 가이드를 추가했습니다.
- 낚시 화면에 지금 해야 할 행동을 보여주는 낚시 코치 패널을 추가했습니다.
- 보이지 않는 레거시 패널 렌더링을 더 줄여 마을 화면 랙을 완화했습니다.

## 테스트
1. 새 브라우저/시크릿에서 접속하면 로그인/시작 화면이 먼저 보이는지 확인.
2. 로컬 게스트 시작 시 Lv.1 / 0G / 호수 시작인지 확인.
3. 기존 데모 상태가 Lv.12 여우로 뜨지 않는지 확인.
4. 모바일 첫 터치 후 주소창 숨김/전체화면 시도가 되는지 확인.
5. 낚시 화면에서 “낚싯대 던지기 → 수면 터치 → 릴 감기” 안내가 보이는지 확인.
