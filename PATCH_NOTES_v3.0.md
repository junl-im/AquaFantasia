# AquaFantasia Patch 19 — v3.0 Ocean Director Mega Update

대형 통합 패치입니다. v2.9 기반에서 UI 과밀도를 줄이고, 원화급 느낌의 SVG 아트팩과 실제 플레이 판단을 돕는 오션 디렉터 패널을 추가했습니다.

## 핵심 변경

- 오션 디렉터 3.0 패널 추가: 추천 행동, 추천 지역, 장비점수, 준비도, 성능 상태를 한 카드에 요약
- 모바일 스마트 독 추가: 낚시, 가방, 장비, 지도 빠른 접근
- v3.0 SVG 아트팩 추가: 배경, 수면광, HUD 패널, 감독관 카드, 지역 카드 텍스처, 낚시 스테이지, 실루엣, 보스 문장, 루트 아틀라스
- 카카오 인앱브라우저 무팝업 방향 유지: 외부 브라우저 권장창 없이 바로 게임 셸 적용
- 첫 사용자 제스처 기반 게임모드 자동 시도 강화
- 모바일 뒤로가기 종료 확인 흐름 유지
- 월드맵 잠금 안내 버그 수정: 현재 장비점수와 필요 장비점수를 분리 표시
- 저장 키 v3.0 추가 및 v2.2/v2.4 호환 저장 누락 보정
- 신규 시그니처 어종 7종 추가
- PWA/Service Worker/Actions 검증 기준 v3.0 갱신
