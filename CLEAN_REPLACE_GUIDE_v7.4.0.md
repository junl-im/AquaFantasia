# v7.4.0 클린 교체 가이드

v7.4.0은 업로드 에셋을 기준 아트팩으로 통합한 버전입니다.

## 권장 적용

1. 저장소 루트에서 `.git` 폴더만 남깁니다.
2. 나머지 기존 파일을 삭제합니다.
3. `AquaFantasia_v7.4.0_USER_ASSET_PACK_FULL_PROJECT.zip`을 압축 해제합니다.
4. GitHub Desktop에서 커밋/푸시합니다.
5. 카카오톡 인앱 브라우저 캐시가 남으면 링크를 완전히 닫고 다시 엽니다.

## 덮어쓰기 패치 사용 조건

v7.3.0을 정확히 클린 적용한 상태라면 덮어쓰기 패치도 가능합니다. 여러 과거 버전이 섞여 있으면 전체 교체를 권장합니다.
