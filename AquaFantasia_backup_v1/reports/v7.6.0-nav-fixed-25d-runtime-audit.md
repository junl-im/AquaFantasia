# Aqua Fantasia v7.6.0 Nav Fixed + 2.5D Runtime Audit

## 핵심 패치
- 하단 네비게이션을 스크롤 화면 내부가 아니라 `#app`의 고정 루트 네비게이션으로 마운트했습니다.
- `position: fixed !important`, 중앙 고정 폭, 높은 z-index, safe-area 대응으로 화면 드래그/스크롤 시 최하단에 유지됩니다.
- 낚시 화면에도 동일 네비게이션을 연결했고, 캐스트 버튼/릴 패널을 네비게이션 위로 재배치해 겹침을 줄였습니다.
- 사용자가 첨부한 v9 PNG 전용 2.5D/3D 렌더드 에셋 중 실제 런타임 우선순위가 높은 UI, 아이콘, 장비, 물고기, FX, 배경을 `public/assets/v9`로 정리했습니다.
- 대형 배경 PNG는 모바일 성능을 위해 1280px WebP로 변환해 PixiJS 런타임 배경으로 연결했습니다.

## 검증 포인트
- `npm run validate`
- `npm run typecheck`
- `npm run build`
- 모바일에서 마을/도감/상점/미션 스크롤 중 하단 네비게이션 위치 고정 확인
- 낚시 화면에서 캐스트 버튼, 릴 패널, 네비게이션 겹침 확인

## 엔진 방향
- 현재 코어는 PixiJS 8 유지가 최선입니다. 이미 Vite/TypeScript/PixiJS/Howler/Firebase/PWA 구조가 잡혀 있고, 2.5D 렌더드 PNG/WebP 중심 게임에는 가볍고 빠릅니다.
- 완전 3D 수중 월드가 목표가 되는 시점에는 PlayCanvas/WebGPU 브랜치를 별도로 실험하는 편이 안전합니다.
