# AquaFantasia v8.2.0 패치 노트

## 핵심 목표

이번 패치는 사용자가 새로 제공한 `AquaFantasia_v13_tab_full_ui_compositions.zip`을 최우선 기준으로 삼아, 기존 불안정한 DOM 카드 UI를 걷어내고 탭별 전체 화면 구성을 실제 런타임에 맞춘 패치입니다.

## 주요 변경

- v13 탭별 전체 UI 구성도 8종을 `public/assets/v13/compositions`에 WebP 런타임 에셋으로 반영했습니다.
  - 마을
  - 낚시
  - 장비
  - 가방
  - 도감
  - 상점
  - 미션
  - 랭킹
- 모든 탭을 v13 기준의 고정형 모바일 게임 화면으로 재구성했습니다.
  - 상단 HUD
  - 본문 패널
  - CTA 버튼
  - 하단 탭 네비게이션
- 이전 DOM 텍스트/카드가 레퍼런스 이미지 글씨와 겹치는 문제를 제거했습니다.
- 보이는 UI는 v13 구성도 이미지를 기준으로 두고, 실제 클릭/터치 영역은 투명 hotspot 버튼으로 별도 배치했습니다.
- 하단 탭 네비게이션을 8탭 구조로 확장했습니다.
  - 마을 / 낚시 / 장비 / 가방 / 도감 / 상점 / 미션 / 랭킹
- 낚시 화면은 v13 구성도 위에 실제 PixiJS 낚시 스테이지를 좌표 기준으로 얹었습니다.
- 시작 화면의 `이 기기에서 로그인 유지` 버튼형 UI는 유지했습니다.
- Vite / TypeScript / PixiJS 8 구조는 유지하면서 화면 안정성에 집중했습니다.

## 이번 패치의 의도

이전 버전은 기능성 DOM을 먼저 배치하면서 레퍼런스 이미지와 실제 텍스트가 충돌했습니다. v8.2.0에서는 반대로, 사용자가 만든 전체 화면 구성도를 먼저 신뢰하고 그 위에 동작만 연결하는 방식으로 바꿨습니다.

즉, 화면은 v13 레퍼런스처럼 보이고, 터치는 실제 버튼으로 작동합니다.

## 검증

아래 명령을 통과했습니다.

```bash
npm run typecheck
npm run validate
npm run build
```
