# AquaFantasia Patch 14 - v2.5 Focus UI Cleanup

## 목표

기능을 많이 넣으면서 복잡해진 화면을 실제 플레이에 필요한 정보 중심으로 정리했습니다.

## 핵심 변경

- 상단 바를 골드/레벨/빠른 메뉴 중심으로 단순화
- 마을 첫 화면을 Focus HUD로 재구성
- 다음 추천 행동 표시: 출석, 보상 수령, 판매, 강화, 낚시
- 사운드/PWA/품질/배포 상태는 접이식 설정 영역으로 이동
- 지역 카드를 작은 카드형으로 정리
- 하단 메뉴는 마을/낚시/장비/가방/미션 중심으로 축소
- 기존 기능은 삭제하지 않고 필요할 때 접근하는 구조로 정리

## 저장 호환

`aqua_v2.5`를 새 저장 키로 사용하며, v2.4 이하 저장 데이터도 계속 읽습니다.
