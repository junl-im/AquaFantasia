# Aqua Fantasia v3.8 Action Fishing UI

## 핵심
- 낚시 화면에 단계별 액션 가이드 추가
- 입질 시 빛나는 터치 타깃 표시
- 릴 감기 패널을 실전형 버튼/수면 터치 구조로 개편
- 마을 UI 겹침 원인이던 스마트 독 기본 숨김
- v37 아트 디렉터 패널을 메인에서 세부 분석 영역으로 이동
- v38 경량 SVG 아트팩 추가
- Service Worker 캐시 v3.8.0 갱신

## 성능
- 숨김 패널 렌더링 억제 유지
- 모바일/카카오 환경에서 무거운 UI 중복 표시 감소
- 액션 낚시 이펙트는 perf-lite에서 자동 약화

## 테스트
1. 첫 화면에서 UI가 겹치지 않는지 확인
2. 낚시 진입 후 `STEP 1` 안내 확인
3. 입질 시 `여기 터치!` 원이 표시되는지 확인
4. 수면 또는 릴 버튼 터치로 포획 진행도가 오르는지 확인
5. 장력 위험 시 안내 문구가 바뀌는지 확인
