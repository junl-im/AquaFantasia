# AquaFantasia v2.1.30

## v2.1.30 변경사항

- v2.1.29의 플레이어 8방향 32프레임과 걷기 모션을 **검증 고정**했습니다.
  - `public/assets/v2129/characters/player`
  - 8방향 × 4프레임 = 32개 PNG 유지
  - 방향 매핑은 파일명 기준 identity 유지
  - 캐릭터 방향 보호 토큰은 유지합니다.
- 일반 NPC 방향 자산을 다시 확인했습니다.
  - `chief`, `merchant`, `guild`, `captain`, `tourist`, `vip`
  - 각 역할별 8방향 PNG가 존재하는지 validate에서 검사합니다.
- 우측 상단 메뉴바를 더 간결하게 정리했습니다.
  - 전체를 감싸던 테두리/배경은 투명화
  - 아이콘 간 간격 확대
  - 아이콘 상단 잔상/잘림 방지 유지
- 우측 하단 메뉴바를 마을 화면에서도 메뉴 페이지와 동일한 기준으로 보이도록 보정했습니다.
  - `홈 / 가방 / 퀘스트 / 지도`
  - 아이콘 크기 소폭 확대
  - 누를 때 밀림/흔들림 방지
- 캐릭터 정보창의 캐릭터명 입력란이 검게 보이는 문제를 수정했습니다.
  - 흰색 아쿠아 입력창
  - 진한 아쿠아 텍스트
  - placeholder 색상 보정
- HUD와 개척 바 사이 간격을 조정했습니다.
- 건물/장식 이름 표기 정책을 정리했습니다.
  - 주요 건물만 이름 표기
  - 장식/오브젝트류 이름 표기는 제거
  - NPC/플레이어 이름 외곽선은 더 얇게 조정
- 건설/이동 UX 1차 개편을 적용했습니다.
  - 건물/장식 선택 시 반투명 프리뷰 유지
  - 원하는 타일을 누르면 중앙 작은 아쿠아 확인창 표시
  - `건설` 또는 `취소` 선택 후 설치/이동 확정
  - 이동 모드도 같은 확인 흐름 사용
- 낚시게임 첫 화면 UI를 1차 개편했습니다.
  - 상단 HUD 겹침 완화
  - 중앙 안내 카드 추가
  - 릴 패널/릴 콘솔/캐스팅 버튼 중앙 정렬 유지
  - 구형 프레임/장식 소품이 화면을 덮지 않도록 유지
- 로그인 유지 토글은 현재 확정 스타일을 계속 고정합니다.
  - 이미지 배경 금지
  - 작은 CSS 아쿠아 토글
  - 선택 상태는 `✓` 표시로만 구분
