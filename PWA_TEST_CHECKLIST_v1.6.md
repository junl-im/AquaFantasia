# Aqua Fantasia v1.6 PWA 테스트 체크리스트

## 배포 후 모바일 테스트
- GitHub Pages 주소를 모바일 Chrome 또는 Safari에서 연다.
- 첫 접속은 온라인 상태에서 진행한다.
- 게임 안의 `📲 설치` 버튼이 보이면 설치를 누른다.
- 설치 버튼이 보이지 않는 브라우저는 브라우저 메뉴의 `홈 화면에 추가`를 사용한다.
- 홈화면 아이콘으로 다시 실행했을 때 주소창이 줄거나 앱 모드로 열리는지 확인한다.

## 오프라인 테스트
- 한 번 정상 접속 후 브라우저를 닫는다.
- 비행기 모드 또는 네트워크 OFF 상태로 전환한다.
- 홈화면 아이콘 또는 동일 URL로 다시 연다.
- 캐시된 화면 또는 `offline.html` 안내가 표시되는지 확인한다.

## 성능 테스트
- 마을 화면의 `⚡ 점검` 버튼을 누른다.
- 50 FPS 이상이면 고급 그래픽 유지.
- 35 FPS 미만이면 자동으로 절전 그래픽으로 전환되는지 확인한다.

## 저장 테스트
- 물고기를 1마리 잡는다.
- 새로고침 후 골드/가방/도감 데이터가 유지되는지 확인한다.
- Firebase 로그인 계정으로 접속한 뒤 클라우드 저장 상태가 표시되는지 확인한다.
