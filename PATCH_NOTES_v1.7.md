# Aqua Fantasia Patch 06 - v1.7 Quest Visual Max

## 핵심

- 데일리 출석 보상 추가
- KST 기준 데일리 미션 리셋 구조 추가
- 데일리 미션 4종 추가: 포획, 캐스팅, 수익, 희귀 어종
- 시즌 미션 5종 추가: 누적 포획, 도감, 수익, 영웅 포획, 전설 포획
- 마을 화면에 오늘의 항해 요약 카드 추가
- 하단 네비게이션에 미션 탭 추가
- View Transition API 지원 브라우저에서 화면 전환 부드럽게 처리
- 낚시 화면 소나 펄스 그래픽 추가
- 미션 카드, 출석 카드, 시즌 카드용 홀로그래픽 UI 추가
- 기존 v1.1 ~ v1.6 저장 데이터 호환 유지, 새 저장 키 aqua_v1.7 추가
