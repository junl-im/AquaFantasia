# Aqua Fantasia v6.9.0 - Start Cleanup, Back Guard & Bite Guide

## 목적
사용자가 제공한 레퍼런스 시작 화면을 유지하되, 실제 게임 시작 전에는 불필요한 메뉴와 유틸리티 버튼이 보이지 않도록 정리했습니다. 또한 모바일 웹게임에서 가장 중요한 뒤로가기 동작을 게임 흐름에 맞게 가로채고, 낚시 입질 시 플레이어가 무엇을 눌러야 하는지 즉시 알 수 있도록 안내 연출을 추가했습니다.

## 시작 화면 정리
- 좌측 상단 공지사항 터치 영역 제거
- 우측 상단 고객센터/설정 터치 영역 제거
- 시작 화면 하단 가방/도감/업적/이벤트/상점 메뉴 제거
- 새 시작 화면 에셋 추가: `public/assets/screens/start_screen_clean_v690.webp`
- 기존 레퍼런스 시작 화면은 보관하되 실제 런타임은 정리된 v6.9 화면을 사용

## 로그인 유지 UI 보정
- `이 기기에서 로그인 유지` 체크박스를 런타임 오버레이로 재구성
- 선택/해제 상태가 명확하게 보이도록 2.5D 체크 박스 추가
- 상태를 `localStorage`에 저장

## 낚시 입질 안내 강화
- 입질 시 중앙에 `물었다! 화면을 눌러 당기세요!` 카드 표시
- 보스 입질은 `보스가 물었다!`로 강조
- 화면 터치 후 릴링에 진입하면 자동으로 안내 카드 제거

## 모바일 뒤로가기 가드
- 낚시터에서 뒤로가기: `마을로 돌아갈까요?` 확인 팝업
- 도감/장비/상점/미션에서 뒤로가기: 마을로 이동
- 마을/시작 화면에서 뒤로가기: `게임을 종료할까요?` 확인 팝업
- 종료 팝업이 떠 있는 상태에서 뒤로가기 한 번 더 누르면 브라우저 이전 페이지 이동 시도

## 카카오톡 인앱 대응 유지
- requestFullscreen/screen.orientation.lock 미호출 정책 유지
- CSS portrait cage 유지
- 회전 안내 팝업 없음

## 검증

- `npm run runtime:check`
- `npm run audit`
- `npm run typecheck`
- `npm run build`
