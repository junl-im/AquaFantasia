# v5.6.1 UI State Cleanup Audit

Screenshot issue reproduced from visual inspection:

- The v5.3 fixed fishing navigator was visible over the first-run guide.
- The legacy smart dock label left `QUICK ACTION` text at the lower-left edge.
- The v5.5.2/v5.5.4 runtime badge showed `Node24 OK / STACK SAFE` to players.

Resolution:

- `src/runtime/v561-ui-state-cleanup.js` adds a production UI sweep.
- The sweep hides `.aqua-nav-v53`, `#v31-smart-dock-label`, `#v30-smart-dock`, `#aqua-v552-badge`, and `.v47-renderer-overlay`.
- The sweep watches DOM changes so older runtime modules cannot reintroduce the labels.
- The PWA cache target is now `aqua-fantasia-v5.6.1-ui-cleanup-20260612`.
