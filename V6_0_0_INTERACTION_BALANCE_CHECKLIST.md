# v6.0.0 Interaction Balance Checklist

## 알림 UX
- [ ] 토스트를 탭하면 닫힌다.
- [ ] 토스트를 좌우로 밀면 닫힌다.
- [ ] 미션/퀘스트/보상/출석 알림 탭 시 미션 화면으로 이동한다.
- [ ] 일반 시스템 알림 탭 시 이동 없이 닫힌다.
- [ ] 네이티브 `alert()` 호출이 모바일 친화 알림 카드로 표시된다.

## 밸런스
- [ ] 입질 창이 너무 길지 않다.
- [ ] 릴 타이밍 커서가 약간 빠르지만 불합리하지 않다.
- [ ] 장력이 예전보다 약간 더 빠르게 오르지만 강화 장비로 대응 가능하다.
- [ ] 보스전은 카운터 타이밍 가치가 유지된다.

## 배경/성능
- [ ] 호수/강/항구/심해/용궁/차원의 바다 배경 위에 수면 레이어가 보인다.
- [ ] 저사양/절전 모드에서는 수면 레이어가 과하게 움직이지 않는다.
- [ ] 기존 v5.7 배경이 가려지거나 글씨 배경으로 돌아가지 않는다.

## 배포
- [ ] `npm run validate`
- [ ] `npm run runtime:check`
- [ ] `npm run renderer:check`
- [ ] `npm run audit`
- [ ] `npm run atlas:check`
- [ ] `npm run clean:report`
