# AquaFantasia v2.1.23

## v2.1.23 변경사항

- 버전을 `2.1.23`으로 동기화했습니다.
- 플레이어 캐릭터 방향/걷기 모션을 다시 고정했습니다.
  - 3시 이동은 실제 3시로 보이도록 `east -> west asset` 매핑을 잠금 처리했습니다.
  - 9시 이동은 실제 9시로 보이도록 `west -> east asset` 매핑을 잠금 처리했습니다.
  - 1시/11시 대각선 보정은 유지했습니다.
  - 8방향 × 4프레임, 총 32프레임을 계속 검증합니다.
  - 걷기 프레임 전환 속도를 더 눈에 보이게 보정했습니다.
- 게임 시작 오프닝 영상은 계속 최초 시작 로딩 전용으로 유지했습니다.
  - 홈, 닫기, 메뉴 복귀, 팝업 종료에서는 오프닝 영상이 다시 연결되지 않아야 합니다.
  - 영상 위를 가리는 구형 큰 테두리/장식 프레임을 다시 차단했습니다.
- `v2123-bug-ui-fishing-stability-root`를 추가했습니다.
  - HUD, 개척 바, 우측 상단 메뉴, 조이스틱, 우측 하단 메뉴의 표시/포인터/정렬 상태를 다시 고정했습니다.
  - 모달/페이지가 열릴 때 레거시 상태 클래스가 UI를 뒤섞지 않도록 정리했습니다.
- 가방, 퀘스트, 지도, 상점, 건설, 개척, 캐릭터 정보창, 건물 정보창, 종료 팝업의 아쿠아 카드 공통 규칙을 다시 보강했습니다.
  - 밝은 아쿠아 카드
  - 카드 바깥 딤/블러
  - 우측 상단 X 닫기
  - 글씨 번짐 방지
  - 중복 닫기 버튼 숨김
  - 메뉴 페이지 중앙 카드 구조
- 캐릭터 정보창의 중복 `가방` 버튼을 제거했습니다.
- 조이스틱은 기본 상태만 중앙이고, 조작 중에는 CSS 변수로 노브가 실제 이동하도록 유지했습니다.
- 낚시 게임 UI를 다시 다듬었습니다.
  - 릴 패널, 릴 콘솔, 감기/풀기 터치존을 중앙 고정했습니다.
  - 낚시 화면에는 마을 하단 메뉴가 붙지 않도록 유지했습니다.
  - 골드/미끼/장비/최근 포획 아이콘 대형화를 계속 제한합니다.
  - 구형 프레임 이미지와 신규 소품이 화면을 덮지 않도록 차단했습니다.
- 로그인 유지 토글은 마음에 든 현재 스타일을 계속 고정했습니다.
  - 이미지 배경을 다시 넣지 않습니다.
  - 작고 얇은 CSS 아쿠아 토글과 `✓` 표시만 사용합니다.
- 캐릭터 방향 계산 함수 자체는 수정하지 않았습니다.
  - `ACTOR_DIRECTION_TEXTURE_FIX`
  - `ACTOR_DIRECTION_TEXTURES`
  - `actorDirectionFromVector`
  - `actorTextureUrl`
  - `actorDirectionQaPasses`
  위 토큰은 계속 검증합니다.

## 검증 기준

- v2.1.23 작업본에서 `npm run validate`가 통과해야 합니다.
- v2.1.23 통파일 ZIP을 새로 풀어 `npm run validate`가 통과해야 합니다.
- v2.1.22 통파일에 v2.1.23 패치 ZIP을 덮어쓴 뒤 `npm run validate`가 통과해야 합니다.
- `package.json`, `package-lock.json`, `APP_VERSION`, service worker cache, offline badge/version이 `2.1.23`으로 동기화되어야 합니다.
- 루트 Markdown은 이 `README.md` 하나만 유지합니다.
- `node_modules`, `dist`, `reports`, 백업 폴더, `.log`, `*_NOTES.md`는 ZIP에 포함하지 않습니다.
- 금지 registry/internal 문자열은 포함하지 않습니다.
