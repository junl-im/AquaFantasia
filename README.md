# AquaFantasia v2.1.17

## v2.1.17 변경사항

- 버전을 `2.1.17`로 동기화했습니다.
- `v2117-layout-input-fishing-root`를 추가해 v2.1.16의 마을 에셋 적용 상태를 유지하면서 UI 위치/입력/낚시 릴 화면을 좁게 보정했습니다.
- `이 기기에서 로그인 유지` 토글은 현재 마음에 든 CSS 아쿠아 토글 상태를 검증 고정했습니다. PNG 배경을 다시 입히지 않고, 선택 상태는 `✓` 체크 표시로만 구분합니다.
- 개척 버튼으로 열리는 화면이 좌측 위/화면 밖으로 밀리는 문제를 막기 위해 개척 보드를 풀스크린 딤/블러 모달 쉘로 만들고, 실제 카드만 중앙에 배치하도록 수정했습니다.
- 우측 하단 `홈 / 가방 / 퀘스트 / 지도` 메뉴바는 누를 때 버튼 폭/위치가 변하지 않도록 `v2117-bottom-nav`와 `right-bottom-no-shift-polish` 규칙으로 고정했습니다.
- 우측 상단 조작바는 요청 순서인 `+ / -`, `건설 / 원점`, `상점 / 출항`을 유지하면서, 아이콘 묶음 카드의 우측 여백을 줄이고 HUD는 그만큼 조금 더 길게 표시되도록 보정했습니다.
- 조이스틱 노브는 기본 상태에서만 중앙이고, 조작 중에는 JS가 CSS 변수 `--v2117-joystick-transform`으로 실제 이동시키도록 변경했습니다.
- 건물 정보 UI는 현재 좋은 방향을 유지하면서 카드 폭/높이, 초상 크기, 상태 뱃지, 본문 줄간격을 정리했습니다. 큰 내부 이미지는 계속 숨겨 정보가 밀리지 않게 했습니다.
- 가방, 퀘스트, 지도, 상점, 건설, 개척, 캐릭정보창, 건물정보창은 공통 아쿠아 카드/페이지 규칙을 다시 묶고 글씨 번짐·겹침을 줄이도록 `text-shadow`, `filter`, 불필요한 stroke를 차단했습니다.
- 모든 열리는 창의 닫기는 우측 상단 원형 `X` 기준을 유지하고, 일반 텍스트 닫기 버튼은 중복으로 보이지 않게 했습니다.
- 낚시게임의 찌/릴 감기/풀기 UI가 좌측으로 밀려 나가지 않도록 `reel-panel`, `v2055-reel-console`, `v2053-reel-touch-zone`, 캐스팅 버튼을 중앙 고정으로 재정렬했습니다.
- 마을 물 타일은 제공한 `public/assets/v2110/tiles_32x32/sea_and_beach`의 `sea_tile_001~006`, `011~015` 계열을 우선 사용하도록 후보를 확장했습니다.
- v2.1.10 신규 에셋 278개는 유지하고, 마을 오브젝트/물 타일/아쿠아 카드 스킨에 제한적으로 사용합니다.
- 캐릭터 방향 고정 로직은 수정하지 않았습니다.

## 유지 규칙

- 루트 Markdown 파일은 `README.md` 하나만 유지합니다.
- `node_modules`, `dist`, `reports`, 백업 폴더, `.log`, `*_NOTES.md`는 ZIP에 포함하지 않습니다.
- 금지 registry 문자열은 포함하지 않습니다.
- `validate-and-deploy` 기준으로 `npm run validate`가 통과해야 합니다.
- 통파일 ZIP과 이전 통파일에 덮어쓰기 가능한 패치 ZIP을 함께 제공합니다.

## 검증 기준

- v2.1.17 작업본에서 `npm run validate`가 통과해야 합니다.
- v2.1.17 통파일 ZIP을 새로 풀어 `npm run validate`가 통과해야 합니다.
- v2.1.16 통파일에 v2.1.17 패치 ZIP을 덮어쓴 뒤 `npm run validate`가 통과해야 합니다.
- `package.json`, `package-lock.json`, `APP_VERSION`, service worker cache, offline badge/version이 `2.1.17`로 동기화되어야 합니다.
