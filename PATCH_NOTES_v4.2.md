# AquaFantasia Patch 31 - v4.2 Result Flow Polish

## 핵심
- 포획 성공 후 결과가 한 번에 우르르 뜨던 흐름을 단계형 완료 화면으로 변경했습니다.
- 포획 확인 → 가치 분석 → 가방 보관 → 선택 버튼 순서로 표시됩니다.
- 기존 대형 폭죽/토스트 연출을 축소하고 모바일 절전 모드에서는 더 가볍게 처리합니다.
- 다시 던지기, 가방, 확인 버튼으로 다음 행동을 직접 선택할 수 있습니다.

## 성능
- 보스 승리 파티클 수를 줄였습니다.
- 일반 포획은 micro burst만 사용합니다.
- 결과 카드 SVG는 1개만 추가하고 Service Worker 핵심 캐시에 포함했습니다.
